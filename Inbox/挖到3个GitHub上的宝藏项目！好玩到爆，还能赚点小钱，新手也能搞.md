## 一、项目一：will-you-be-my-valentine-react —— 你点不到的"拒绝按钮"

💝 一个让"拒绝"按钮会逃跑的表白网页。页面上那个"拒绝"按钮，它会跑！鼠标一靠近就溜走，你怎么都点不到。配上肉麻文案、动态小动画和表情包，氛围拉满。纯前端，不用后端不用数据库，代码特别轻，手机电脑都自适应。

![](https://relay-1.bijitongbu.site/p/543c7a2b1838cc19152f99a3db491430.png)

## 适用场合：

-   给对象或者暗恋的人整活：情人节、520、七夕、生日，都比发红包有意思
    
-   发朋友圈/群聊/抖音：让朋友帮你点，传播性贼强
    
-   学生党整蛊：宿舍之间开玩笑，或者做个恶搞链接
    
-   小商家引流：奶茶店、花店、情侣礼品店，搞个专属页面吸粉
    

## 部署方法：

```
git clone https://github.com/Navaneeth223/will-you-be-my-valentine-react.git
cd will-you-be-my-valentine-react
npm install
npm run start  # 跑起来看看效果
npm run build  # 打包成静态文件
```

打包完直接扔到[[Netlify]]、[[Vercel]]（免费）、或者[[阿里云OSS]]上就行，**服务器一分钱不用花**。

**能怎么改？**

-   换文案：表白的话、标题、按钮上的字，随便改
    
-   换背景：换成你们的合照、或者节日海报
    
-   做多套模板：搞一个"情人节版"，再来个"生日版"
    
-   加小功能：让对方输入自己的名字、随机生成土味情话、加背景音乐开关

## 二、项目二：op7418/guizang-ppt-skill —— PPT里藏彩蛋的"隐身术"

🎭 帮PPT加"隐藏内容+点击解密"效果的工具。你不需要手动做复杂动画，一键就能让文字"隐身"，然后点一下慢慢显现出来。不管是老师讲课藏答案，还是职场人汇报时展示敏感数据，都特别实用。

![](https://relay-1.bijitongbu.site/p/a6254cfaa12713b9a9f7ad83f6ae0281.png)

**适合谁用？**

-   老师：课件里藏知识点，课堂上让学生点开解密，互动感强
    
-   打工人：汇报PPT里藏一些关键数据，讲到哪放到哪
    
-   培训师：公开课或者企业内训，做分步演示
    
-   学生答辩：毕设PPT加点特效，比别人的高级一截
