都说 [[Agent]] 时代，taste 品味是唯一重要的事情，那到底什么是关于什么的品味呢

比如我是程序员，我就该跟 ai 持续，长久是去探讨这三件事儿：

①什么是好的软件，什么是 agent 时代好的软件（下定义）

②如何构建好的软件呢？如何让 agent 去构建好的软件（怎么做）

③我该围绕这件事儿积累哪些 skill （做什么）

同理，如果你是其他职业，也可以把好的软件，换成好的“xx”，然后把这三个问题一同丢给 ai
