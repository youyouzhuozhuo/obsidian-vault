## 一、Agent UI（字节开源·桌面自动化AI）：动动嘴，电脑自己干活

## 项目名片

-   开源方：[[字节跳动]]
    
-   GitHub热度：今日涨星+656，总星32k+
    
-   一句话定位：**能"看懂"屏幕的[[多模态]]AI桌面助手**——你说人话，它帮你点鼠标。
    

### 1. 零代码，打字/说话就能指挥电脑

不用学编程，不用记快捷键，直接跟它唠：

-   "打开VS Code，把自动保存延迟改成500ms"→它自己点菜单、搜设置、改参数，一气呵成
    
-   "打开浏览器，搜'2026年热门电影'，点第一个链接"→全程自动操作，你嗑瓜子就行
    
-   "把桌面上所有图片挪到'照片'文件夹"→自动拖拽、分类，强迫症福音
    

![[笔记同步助手/images/75548a17ca32db63436a2495f2873168_MD5.png]]

### 2. 真·视觉识别，不靠死坐标

传统[[RPA]]一换界面就废，Agent UI用的是字节自研**[[UI-TARS视觉模型]]**，直接"看"屏幕截图，按钮、输入框、菜单一眼识别，界面改版也不怕。比那些靠坐标硬编码的灵活十倍。

​

### 3. 全平台+本地跑，隐私安全不操心

-   支持Windows/macOS，桌面、浏览器、文件管理全都能控
    
-   数据**本地处理，不上云**，隐私有保障
    
-   开源免费，商用也行，GitHub直接下，即装即用
    

**上手难度：⭐⭐**（新手10分钟搞定）

```
# 极速上手三步走
git clone https://github.com/agent-ui/AgentUI
npm install
npm start    # 泡杯茶看它表演
```

## 二、moviemon（终端追剧神器）：装个逼，命令行里追全网剧

## 项目名片

-   开发语言：[[Python]]
    
-   GitHub热度：小众黑马，最近极客圈口碑炸裂
    
-   一句话定位：**纯终端影视工具**，没界面、没广告，一行命令搜片看详情
    

### 1. 搜片+详情+播放，全在命令行搞定

不用开浏览器、不用点APP，终端里直接敲：

-   搜片：`moviemon --search "流浪地球2"`→秒出全网结果，[[IMDb]]评分、导演、演员全都有
    
-   按分类找：`moviemon --popular`（热门）、`moviemon --genre sci-fi`（科幻）
    
-   看详情：`moviemon --id 27205`→剧情简介、上映年份、获奖情况一目了然
    

### 2. 轻量无广告，老电脑也能跑

-   只要Python环境，**内存占用不到10MB**，十年前的老爷机照样丝滑
    
-   支持Linux/macOS/Windows，终端颜色、参数随便调
    
-   调用[[TMDb]]权威API，数据实时更新，**没有弹窗、没有会员套路**
