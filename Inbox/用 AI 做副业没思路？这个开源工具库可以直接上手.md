很多人用 AI 做副业，最大的问题不是不会用工具，而是每次都从零开始想。

写短视频脚本，要重新想框架。

写小红书文案，要重新想开头。

做产品介绍，要重新想卖点。

写公众号文章，要重新调 [[Prompt]]。

时间就这样浪费掉了。

最近看到一个 [[GitHub]] 上很火的开源项目 [[prompts.chat]]，可以把它理解成一个现成的 AI Prompt 工具库。

它以前叫 [[Awesome ChatGPT Prompts]]，里面沉淀了大量角色提示词、内容创作模板、营销文案模板和产品策划模板，对做内容、自媒体、副业、产品推广的人来说，确实很实用。

它最大的价值，是帮你把很多重复性工作标准化。

比如你要做短视频，就可以直接找脚本、标题、钩子、分镜相关的 Prompt。

你要写社媒内容，就可以找帖子、线程、产品介绍、种草文案相关的 Prompt。

你要做数字产品，就可以用它辅助生成选题、卖点、落地页文案、定价思路和销售页面。

你要做博客或 [[SEO]] 内容，也可以用它生成文章结构、FAQ、关键词布局和内容优化建议。

对普通人来说，这类工具最适合的场景，其实不是幻想一夜暴富，而是把自己原来很慢的工作变快。

以前写一篇内容可能要两三个小时，现在先用模板跑出初稿，再自己改案例、改口吻、改观点，效率会高很多。以前做一个产品介绍页，要反复想标题、痛点、卖点、行动引导，现在可以先让 AI 给出多个版本，再挑一个方向继续打磨。

更适合副业玩家的用法，是把它接进自己的固定流程里。

比如做小红书，可以固定成“选题分析、爆款拆解、标题生成、正文初稿、封面文案、评论区回复”这几步。

比如做知识付费，可以固定成“课程选题、用户痛点、课程大纲、销售页、成交话术、交付资料”这几步。

比如做企业服务，可以固定成“客户行业分析、账号诊断、方案包装、报价说明、案例文案、跟进话术”这几步。

这样用 AI，才更像一个副业系统。

prompts.chat 还支持 [[MCP]]，可以接入 [[Cursor]]、[[VS Code]]、[[Claude Code]]、[[Claude Desktop]] 这类工具。

对有一点技术基础的人来说，可以把它变成工作流的一部分，让 AI 在写代码、做内容、做营销方案的时候，直接调用对应模板。这样就不用每次手动复制粘贴 Prompt。

不过要说清楚，Prompt 只是提效工具，不会自动替你赚钱。

真正能变现的，还是你选择了什么赛道，解决了谁的问题，能不能持续交付结果。

如果你想用 AI 做副业，可以先从三个方向切入。

内容型副业：短视频、小红书、公众号、知乎、博客，用 AI 提高内容产出速度。

服务型副业：帮企业写文案、做账号诊断、做宣传方案、做 AI 工作流搭建。

产品型副业：做模板包、资料包、电子书、课程、[[Notion]] 系统、行业 Prompt 包。

普通人更建议从服务型副业开始，因为它离成交最近。

比如帮本地商家整理企业介绍、写门店推广文案、做短视频脚本、搭建公众号内容库、设计小红书获客流程。这些事情客户能看懂，也更愿意付费。等你服务过几个客户，再把重复交付的部分整理成模板包或课程，后面才更容易做产品化。

所以，AI 副业的核心思路很简单。

介绍开源项目 prompts.chat（原 Awesome ChatGPT Prompts），提供大量现成 Prompt 模板，可提升内容创作和副业效率，并建议从服务型副业切入。

### 可信度判断

来源为微信公众号，但引用的 GitHub 项目真实存在且知名，内容逻辑合理，无夸大或虚假信息。需核实 prompts.chat 是否确实支持 MCP 及与 Cursor 等工具的集成。

### 可利用价值

直接提供可复用的 Prompt 模板库，减少从零开始的时间；给出副业切入方向（内容、服务、产品），尤其推荐服务型副业，实操性强。

可开发一个基于 prompts.chat 的 MCP Server，集成到 Claude Code 或 Cursor 中，实现一键调用模板；或制作特定领域（如小红书、短视频）的 Prompt 包作为数字产品出售。
