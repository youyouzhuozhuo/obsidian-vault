上个月我用 [[Claude Code]] 做了个 [[雅思]] Skill，发了条使用教程到 [[小红书]]，20 多万人看。

评论区和私信炸了，全是要 Skill 的。我建了 5 个 500 人粉丝群，很快满员。

免费分享一阵后我在想，能不能把它升级成可视化教练系统，挂橱窗卖。

定价 99 元，第一天 10 个人买，还有 2 个定制客户。现在 200 多人买了，三个平台 10 天涨粉 2 万，加上一对一咨询、Claude Code 部署、定制 Skill，20 天变现 4 万。

今天把完整流程拆给你，任何人都能复制。

这个生意的起点就一句话——

找一个受众大的、需求够硬的方向，做低客单价的虚拟产品。

举几个：雅思备考模板、[[Notion]] 模板、大神思维模型整理、[[TED]] 演讲稿整理。

售价 39-99 之间，不要超过 99。

为什么不能超过 99？超过 99 客户就开始动脑子了——「我真的需要吗」「我能不能自己做」。99 以下基本是冲动消费区，对的人看到价格直接就付了。

第一步，选你要做什么。

如果你不知道做什么，就从自己上学到上班划过的钱开始捋。你付过的所有课程、训练营、模板、付费社群，每一笔都是潜在题材。你自己付过钱，意味着这个需求是真的。

随便选一个，去小红书搜，看有没有产品。有的话买一份。

然后 Claude Code 上场。

把你买回来的产品扔给 cc，让它洗稿一遍，包装成你自己的。

最好申请一个版权证明，学猫猫侠那样。然后上架小红书橱窗。

这一步最麻烦，下期单独写一篇讲。

产品有了，下一步铺内容。

回到你刚才买产品的那家店。这家店肯定关联了一些博主橱窗，也有自己的账号。看他发什么，直接拿去给 Claude Code 洗稿，发出来。

要像素级模仿，但不要纯搬运。

模仿和搬运的边界在哪，后面专门出一期讲。先记一句：模仿结构和角度，内容、案例、表达全部换成你自己的。

cc 把这件事的效率拉得非常高。纯图文一天能出 5-10 篇，三天做完一周的量。

剩下的就是按时出摊等爆款，每天花的时间少于一小时。

然后是交付。

不要直接发文件。给客户一个网盘链接。

两个原因：

第一，[[百度网盘]]和[[夸克网盘]]都有拉新返现，客户为了下载会注册，你顺带赚一笔。150 单算下来多出几百到上千块。

第二，更重要——网盘文件夹里除了客户买的产品，要放上你其他产品的介绍，还有微信二维码。

为什么？

因为我们要的是一个链路收钱，不是一笔生意。

虚拟产品只是付费的开始，不是付费的结束。

后面要跟上的东西：

定制服务、供应链、其他产品、用这个产品配套要的工具。
