卸载龙虾居然也需要收费了，那我就把市场价打下来，学会了你也可以去帮忙卸载龙虾赚钱。
卸载龙虾只需要三步：
1.openclaw gateway status 确保是stopped
然后 openclaw gateway stop
停止Gateway服务
2. openclaw uninstall
3. npm rm -g openclaw
遇到权限问加上sudo
卸载好之后检查一下有没有卸载干净
npm list -g openclaw
ls ~/.openclaw/
确保是返回空
就卸载干净啦
