内容：
试了300多个[[OpenClaw]]技能，90%都是花架子——真正值得装的就这10个
技能市场越来越热闹，但大部分Skills装完你会发现：没啥用，还吃Token。与其装100个吃灰，不如把这10个核心的配好，覆盖养龙虾的所有刚需场景。
【记忆与成长】
1️⃣lossless-claw——给龙虾装长期记忆。对持久化存数据库，后台自动压缩成树状摘要控制Token消耗。装这个等于给龙虾换了个脑子，不会每次对都失忆。
npx skills add Martian-Engineering/lossless-claw
2️⃣self-improving-agent——让龙虾学会复盘。自动记录你纠正过的错误、踩过的坑、反复提到的习惯，整理成"知识卡片"后续任务主动调用。不会复盘的龙虾，养多久都是新手。
npx skills add charon-fan/agent-playbook
【安全防线】
3️⃣skill-vetter——装新技能前先安检。扫描有没有网络外发、读敏感环境变量、写系统目录、可疑的base64解码。技能越装越多，风险也越大。
npx skills add useai-pro/openclaw-skills-security
4️⃣openclaw-backup——最后一道保险。定时备份配置文件和记忆库，养死了也能一键恢复，配置、记忆、历史对全部回档。
npx skills add theagentservice/skills
【自动化操作】
5️⃣bb-browser——免登录神器。直接调用本地浏览器的登录态，龙虾用你的身份操作，不用每次扫码。
npx skills add epiral/bb-browser
6️⃣agent-browser——让龙虾替你操作网页。说一句"去某某网站把某某列表抓下来"，自己开浏览器、操作页面、结构化返回数据。[[Vercel]]出品。
npx skills add vercel-labs/agent-browser
7️⃣OpenCLI——18个平台一句操作。[[B站]]、知乎、小红书、X、Reddit、GitHub……评论、关注、搜索全部做成命令行接口。
npx skills add joeseesun/opencli-skill
8️⃣cron-claw——给龙虾装定时任务。每天早上8点抓竞品动态、每周五自动生成周报、每隔4小时刷一遍关键词舆情——设好时间和指令，不用盯着。
