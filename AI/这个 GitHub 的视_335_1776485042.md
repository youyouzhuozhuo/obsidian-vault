这个 GitHub 的视频自动剪辑工具 [[AutoClip]]，我先 fork 了。
完全就是一个 AI 剪辑师。
丢一个 [[YouTube]] / [[B站]]视频链接进去，
AI 会自己：
-下载视频
- 分析内容
- 找出精彩片段
- 自动剪成多个短视频
- 甚至顺手把标都写好
也就是说——
一条长视频 → 自动变成一堆短视频素材。
特别适合：
- 直播回放拆短视频
- 播客剪高光
- 访谈剪金句
- 课程拆知识点
做内容矩阵的人，应该一眼就懂这个价值。
很多人做短视频最累的其实不是拍，
而是 一点点拖时间轴剪片段。
这个项目就是把这一步
直接全自动化了。
