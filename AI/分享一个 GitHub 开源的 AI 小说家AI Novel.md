AI_Novel 是一款基于大语言模型（[[LLM]]）构建的智能内容生成系统，核心定位是**自动化生成小说 / 课程内容的智能写作助手**，提供完整的 Web 交互界面和全流程内容管理能力，遵循 MIT 许可证（LICENSE 文件标注 Apache 2.0，文档标注 MIT），适配 Python 3.7+ 环境。
