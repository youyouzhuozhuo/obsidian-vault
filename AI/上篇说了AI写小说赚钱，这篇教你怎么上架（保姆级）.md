# ▎先纠正一个误区

很多人以为写完小说上传就完了。

不是的。​海外网文平台是「签约制」，你得先过编辑那一关。

流程是这样的：注册作者账号 → 上传前5-10章 → 等平台审核 → 编辑看中了给你发签约邀请 → 签约后平台给你推流量 → 读者付费看你的章节 → 平台跟你分成。

没签约之前，你的书基本没有曝光。所以​前10章的质量，决定了你能不能拿到入场券。

​

> 【金句】海外网文赚钱的门槛不是写作能力，是前10章能不能打动编辑。

---

# ▎第一步：选平台，别选错

我试过3个平台，直接给你结论：

[[GoodNovel]]​—— 新手首选。审核门槛低，编辑回复快，分成比例还行（50%-70%看合同类型）。最关键的是，它对非英语母语作者非常友好，你英文不完美也能过。

[[Dreame]]​—— 流量大，但签约要求高一些。适合第二本书投。它和GoodNovel是同一家公司（[[Stary Group]]），但编辑团队不同。

[[Webnovel]]（起点国际）​—— 大平台，竞争也大。新手不太建议一上来就投这里。

我的建议：先在GoodNovel上架第一本，跑通整个流程，再往其他平台铺。

​

---

# ▎第二步：前10章，生死局

编辑看你的书，就看前面几章。所以这10章得精心打磨。

我第一本书的类型是 Billionaire Romance（霸总），这是平台最吃香的类型，没有之一。

我的大纲是这样做的：

先用 [[Claude]] 帮我头脑风暴，Prompt 是这样写的：

「我要写一本面向GoodNovel平台的Billionaire Romance小说。女主是普通打工人，男主是隐藏身份的亿万富翁。请帮我设计5个不同的故事开局，要求每个开局都有强冲突和悬念。」

AI 给了我5个方案，我挑了一个最有冲突感的：​女主被渣男未婚夫当众羞辱退婚，一怒之下跟旁边一个陌生男人领了证，结果这个男人是全城首富。

老套？没错。但在海外平台，​老套 = 验证过的爽点 = 读者愿意付费。

​

> 【金句】别在赚钱的事情上追求创新。读者掏钱买的是情绪，不是文学奖。

---

# ▎第三步：AI 扩写的实操细节

这步很多人问得最多，我讲细一点。

我用的工具是 Claude，不是 [[ChatGPT]]。​原因很简单：Claude 写长文的连贯性比 GPT 好很多，人物对话更自然，不容易出现那种一看就是AI的塑料感。

我的 Prompt 模板（直接抄）：

```
你是一个专业的英文网络小说作者，擅长写 Billionaire Romance，文风参考 [[Wattpad]] 和 GoodNovel 上的热门作品。

背景设定：[粘贴你的世界观和人物设定]

上一章摘要：[粘贴上一章最后3段]

本章大纲：[你的200字中文大纲]

要求：
1. 扩写成2500-3000词的英文章节
2. 必须包含至少3段对话
3. 要有心理描写，让读者能感受到女主的情绪
4. 章节结尾必须是一个 cliffhanger（悬念钩子）
5. 语言要通俗，面向ESL读者（英语非母语读者）
6. 不要用太复杂的词汇，目标阅读水平是高中英语
```

关键细节：每次都要把上一章的结尾喂给 AI。​不然角色会突然失忆，读者分分钟弃书。
