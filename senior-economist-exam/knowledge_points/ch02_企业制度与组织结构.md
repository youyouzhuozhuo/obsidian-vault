# 第二章 企业制度与组织结构

> **分值占比：约10% | 论述题+案例分析题常考**
> **目标：60分万岁，抓大放小，以下标注出题概率**

---

## 第一节 企业制度

### 一、企业制度的概念 ★★★

**核心定义**：企业制度是指以产权制度为核心的企业组织形式和经营管理制度。

**企业制度的核心** = **产权制度**

### 二、企业制度的三种基本类型 ★★★★★

**必考对比表**：

| 比较维度 | 个人独资企业 | 合伙制企业 | 公司制企业 |
|---------|-----------|----------|----------|
| **出资人** | 一个自然人 | 两个或两个以上合伙人 | 多元股东（有限责任公司≤50人，股份有限公司无上限） |
| **法律责任** | **无限责任** | **无限连带责任**（普通合伙）/ 有限责任（有限合伙人） | **有限责任**（以出资额为限） |
| **财产归属** | 归业主个人所有 | 归合伙人共有 | 归公司法人所有 |
| **所有权与经营权** | 合一 | 基本合一 | **分离**（核心优势） |
| **筹集资金** | 困难 | 较困难 | 容易（尤其股份有限公司） |
| **设立难度** | 简单 | 较简单 | 复杂 |
| **税收** | 个人所得税 | 个人所得税（先分后税） | **双重征税**（企业所得税+股东个人所得税） |
| **存续性** | 随业主死亡而终止 | 随合伙人变动可能终止 | **永续经营** |
| **典型适用** | 小型零售、服务业 | 律师/会计师事务所等 | 各类规模企业 |

> **考试重点**：公司制企业的两大核心特征 = "有限责任" + "所有权与经营权分离"

### 三、有限责任公司 vs 股份有限公司 ★★★★

| 比较维度 | 有限责任公司 | 股份有限公司 |
|---------|-----------|------------|
| **股东人数** | 50人以下 | 2人以上，无上限 |
| **资本划分** | 不划分为等额股份 | 资本划分为等额股份 |
| **出资证明** | 出资证明书 | 股票 |
| **股权转让** | 向股东以外的人转让需过半数同意 | 可自由转让（上市公司的股票可公开交易） |
| **设立方式** | 发起设立 | 发起设立或募集设立 |
| **信息披露** | 无须公开 | 上市公司须公开披露 |
| **组织机构** | 较简化（可只设执行董事、1-2名监事） | 完整的法人治理结构 |
| **适用范围** | 中小企业 | 大型企业、上市公司 |

### 四、现代企业制度 ★★★★

**核心特征（四自一独）**：
1. **产权清晰** -- 明确出资者所有权与企业法人财产权
2. **权责明确** -- 出资者、经营者各方权责分明
3. **政企分开** -- 政府不直接干预企业经营
4. **管理科学** -- 科学的法人治理结构和管理制度

**现代企业制度的核心** = **公司法人治理结构**

## 第二节 公司法人治理结构

### 一、法人治理结构的组成 ★★★★★

**三大机构+关系图**：

```
  股东（大）会 ──── 最高权力机构
       |
       | 选举/更换
       v
    董事会 ──── 决策机构（对股东会负责）
       |
       | 聘任/解聘
       v
    经理层 ──── 执行机构（对董事会负责）

  监事会 ──── 监督机构（对股东会负责）
    （监督董事会和经理层）
```

### 二、股东（大）会的职权 ★★★★★

**必背核心职权**：

| 类别        | 具体职权                                         |
| --------- | -------------------------------------------- |
| **根本性决策** | 决定公司的经营方针和投资计划                               |
| **人事任免**  | 选举和更换非由职工代表担任的董事、监事                          |
| **重大事项**  | 修改公司章程；增减注册资本；合并/分立/解散/清算/变更公司形式             |
| **财务审批**  | 审批董事会、监事会的报告；审批年度财务预算方案、决算方案；审批利润分配方案和弥补亏损方案 |

**特别决议事项（必须经2/3以上表决权通过）** ★★★★★：
1. 修改公司章程
2. 增加或减少注册资本
3. 公司合并、分立、解散
4. 变更公司形式

> **考试口诀**："章程资本加减、合分解散变形式"

### 三、董事会的职权 ★★★★

**必背核心职权**：

| 类别 | 具体职权 |
|------|---------|
| **执行** | 执行股东（大）会的决议 |
| **经营决策** | 决定公司的经营计划和投资方案 |
| **人事** | 决定公司内部管理机构的设置；聘任/解聘公司经理；根据经理提名，聘任/解聘副经理、财务负责人 |
| **制度** | 制定公司的基本管理制度 |
| **方案制定** | 制定年度财务预算方案、利润分配方案、增减注册资本方案等 |

**董事会会议规则** ★★★：
- 董事会会议应有**过半数**董事出席方可举行
- 董事会作出决议须经**全体董事的过半数**通过

### 四、经理层的职权 ★★★★

经理对董事会负责，行使以下职权：
1. 主持公司的生产经营管理工作
2. 组织实施董事会决议
3. 组织实施公司年度经营计划和投资方案
4. 拟订公司内部管理机构设置方案
5. 拟订公司的基本管理制度
6. 制定公司的具体规章
7. 提请聘任/解聘公司副经理、财务负责人
8. 聘任/解聘除应由董事会聘任/解聘以外的管理人员

> **董事会与经理层的职权对比**：董事会"决定" → 经理层"拟定"并"执行"。注意用词区别。

### 五、监事会的职权 ★★★★

**必背核心职权**：

| 类别       | 具体职权                                        |
| -------- | ------------------------------------------- |
| **财务监督** | 检查公司财务                                      |
| **行为监督** | 对董事、高级管理人员执行职务的行为进行监督；对违法/违规/违章的董事、高管提出罢免建议 |
| **纠正权**  | 当董事、高管的行为损害公司利益时，要求其予以纠正                    |
| **会议权**  | 提议召开临时股东（大）会；在董事会不履行召集和主持股东会职责时召集和主持股东会     |
| **提案权**  | 向股东（大）会提出提案                                 |
| **诉讼权**  | 依照《公司法》规定，对董事、高管提起诉讼                        |

**监事会组成要求** ★★★：
- 监事会应当包括**股东代表**和**适当比例的职工代表**（职工代表比例不低于1/3）
- 董事、高级管理人员**不得兼任**监事

### 六、独立董事制度 ★★★

| 要点 | 内容 |
|------|------|
| **定义** | 独立于公司股东且不在公司内部任职的董事 |
| **任职条件** | 与公司无重大利益关系 |
| **核心作用** | 保护中小股东利益 |
| **适用** | 上市公司必须设独立董事 |

## 第三节 中国特色的国有独资公司法人治理结构

### 一、国有独资公司的特殊性 ★★★★★

**2022年真题考点！高频中的高频！**

**国有独资公司**：国家单独出资、由国务院或者地方人民政府授权本级人民政府国有资产监督管理机构履行出资人职责的有限责任公司。

**特殊之处**：
1. **不设股东会** -- 由国有资产监督管理机构行使股东会职权
2. 可授权董事会行使股东会的**部分**职权（注意是"部分"而非"全部"）
3. 必须设立**党委会（党组）**

### 二、党委会、董事会、经理层的职责 ★★★★★

**必背三大机构职责定位对比（2022年真题原题！）**：

| 治理主体 | 职责定位（九字诀） | 核心职责 |
|---------|----------------|---------|
| **党委（党组）** | **把方向、管大局、保落实** | 领导核心和政治核心作用；保证党和国家方针政策在企业的贯彻执行；对重大事项进行前置研究讨论 |
| **董事会** | **定战略、作决策、防风险** | 重大决策权；选人用人权（对经理层的聘任/解聘）；薪酬分配权；年度工作报告制度 |
| **经理层** | **谋经营、抓落实、强管理** | 主持生产经营管理；执行董事会决议；对董事会授权范围内的事项自主决策 |

**治理原则** ★★★★：
> **"不缺位、不越位，不相互替代、不各自为政"**

### 三、国有独资公司董事会特殊规定 ★★★

| 要点 | 具体内容 |
|------|---------|
| **组成** | 董事会成员中应当有**公司职工代表** |
| **任期** | 每届任期不得超过**三年** |
| **董事长** | 由国有资产监督管理机构从董事会成员中**指定** |
| **外部董事** | 应当有**外部董事**（且外部董事原则上应占多数） |
| **重大事项** | 重大决策须经党委会**前置研究讨论**后，再由董事会决策 |

### 四、党委会前置研究 ★★★★

**核心流程**：
```
党委会前置研究讨论 --> 董事会决策 --> 经理层执行
```

**党委会前置研究讨论的重大事项范围**：
1. 贯彻执行党中央决策部署的重大举措
2. 企业发展战略、中长期发展规划
3. 企业重要改革方案
4. 企业重大投资项目
5. 重要人事任免
6. 大额资金运作

## 第四节 企业组织结构

### 一、组织结构的基本类型 ★★★★★

**必考！四种基本组织结构对比表**：

| 比较维度 | U型（直线职能制） | M型（事业部制） | H型（控股型） | 矩阵制 |
|---------|-----------------|---------------|-------------|--------|
| **全称** | Unitary（一元型） | Multidivisional（多事业部型） | Holding（控股型） | Matrix（矩阵型） |
| **权力特征** | **高度集权** | 集权+分权 | **高度分权** | **双重领导** |
| **划分依据** | 职能 | 产品/地区/客户 | 产权（股权） | 职能+项目 |
| **法律地位** | 无法人资格 | 无法人资格 | **子公司独立法人** | 无法人资格 |
| **管理关系** | 垂直行政指挥 | 总部-事业部 | 母子公司产权关系 | 职能经理+项目经理 |
| **协调方式** | 自上而下 | 总部协调+事业部自主 | 市场化运作 | 双向交叉协调 |
| **灵活性** | 低 | 中 | 高 | 高 |
| **适用规模** | 中小型 | 大型多元化 | 集团型企业 | 项目驱动型 |
| **优点** | 指挥统一、职责清晰 | 适应多元化、市场响应快 | 资本运作灵活 | 资源利用率高、灵活 |
| **缺点** | 横向协调难、高层负荷重 | 资源可能重复配置 | 控制力较弱 | 双重领导冲突、管理复杂 |
| **首创者** | - | **斯隆**（通用汽车） | - | - |

> **记忆口诀**：U=统一（集权），M=多事业部（斯隆），H=控股（产权），矩阵=两条线交叉

### 二、组织结构的具体形式 ★★★★

#### 1. 直线制 ★★★

```
总经理
  |
  |--- 车间主任 --- 班组长
  |--- 车间主任 --- 班组长
```

- 最简单的组织形式
- 特点：没有职能部门，命令由上至下直线传递
- 优点：命令统一、决策迅速
- 缺点：管理者负担过重，缺乏专业化管理
- 适用：小型企业

#### 2. 职能制 ★★★

- 各职能部门有权在自己的业务范围内向下级下达命令
- 优点：专业化管理
- 缺点：多头领导，命令不统一
- 实际中很少单独采用

#### 3. 直线职能制（U型） ★★★★★

```
           总经理
          /      \
   职能部门    车间/业务部门
  (参谋角色)    (直线指挥)
```

- 直线部门拥有指挥权，职能部门只起参谋作用
- **最常见**的组织形式
- 缺点：横向协调困难、适应性差

#### 4. 事业部制（M型） ★★★★★

```
          总部
         /  |  \
    事业部A 事业部B 事业部C
    (利润中心，独立核算)
```

- "集中政策、分散经营"
- 事业部是**利润中心**
- 总公司保留：重大决策权、人事权、财务控制权
- 事业部享有：日常经营自主权
- 适用：大型多元化企业

#### 5. 矩阵制 ★★★★

```
          项目A  项目B  项目C
           |      |      |
职能部门 --+------+------+--
           |      |      |
```

- 员工同时受职能部门经理和项目经理双重领导
- 优点：打破部门壁垒、灵活应变
- 缺点：双重领导冲突
- 适用：需要跨部门协作的项目型组织

### 三、N型结构（网络型结构） ★★★

| 要点 | 内容 |
|------|------|
| **定义** | 以合同为基础，依靠外部机构进行制造、分销等活动的组织形式 |
| **核心** | 企业只保留核心功能，非核心业务外包 |
| **特点** | 极度精简、高度灵活 |
| **适用** | 知识密集型、快速变化行业 |

## 第五节 企业组织变革与创新

### 一、组织变革的动因 ★★★★

| 动因类别 | 具体内容 |
|---------|---------|
| **外部环境** | 技术、市场、竞争、政策法规变化 |
| **内部因素** | 战略调整、规模变化、管理问题、组织文化冲突 |
| **技术驱动** | 数字化转型、信息化建设 |

### 二、组织变革的方向 ★★★★★

**四大方向（必背）**：

| 方向 | 含义 | 关键词 |
|------|------|--------|
| **扁平化** | 减少管理层次、增加管理幅度 | 去中间层、缩短决策链 |
| **柔性化** | 增强组织适应性和灵活性 | 动态调整、快速响应 |
| **网络化** | 打破组织边界，构建内外协作网络 | 跨组织合作、虚拟团队 |
| **合作化** | 强化团队协作和跨部门协同 | 打破部门墙、团队化运作 |

> **记忆口诀**："扁柔网合"（扁 = 扁平化，柔 = 柔性化，网 = 网络化，合 = 合作化）

### 三、组织变革的过程 ★★★★

**勒温的三阶段变革模型**：

```
解冻期（打破旧状态） --> 变革期（实施变革） --> 再冻结期（巩固新状态）
```

| 阶段 | 核心任务 |
|------|---------|
| **解冻期** | 创造变革动力，打破旧有习惯和阻力 |
| **变革期** | 实施新的行为、结构和流程 |
| **再冻结期** | 稳定变革成果，防止回退 |

### 四、组织变革的阻力及对策 ★★★

| 阻力来源 | 表现 | 应对策略 |
|---------|------|---------|
| **个体阻力** | 习惯、安全感、经济因素、对未知的恐惧 | 沟通、培训、参与 |
| **组织阻力** | 结构惯性、专业群体的威胁、既得利益 | 教育沟通、参与决策、支持与承诺 |
| **文化阻力** | 价值观冲突、传统思维 | 文化建设、领导示范 |

### 五、组织创新 ★★★★

| 创新类型 | 内容 |
|---------|------|
| **结构创新** | 组织结构的重新设计（如从U型转向M型） |
| **流程创新** | 业务流程再造（BPR） |
| **文化创新** | 培育创新文化 |
| **数字化转型** | 利用数字技术推动组织变革 |

### 六、学习型组织 ★★★

**彼得-圣吉的"第五项修炼"五项修炼**：

| 序号 | 修炼内容 | 核心含义 |
|------|---------|---------|
| 1 | **自我超越** | 个人持续学习和成长 |
| 2 | **改善心智模式** | 打破固有思维定式 |
| 3 | **建立共同愿景** | 共享的目标和价值观 |
| 4 | **团队学习** | 团队协作与知识共享 |
| 5 | **系统思考** | 从整体视角看问题（核心修炼） |

> **注意**：第五项修炼 = 系统思考，是五项修炼的核心和基石
