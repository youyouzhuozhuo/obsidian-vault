# 第五章 质量管理与安全管理

> 考试权重：**5%** | 优先级：较低 | 本章以选择题和案例分析小问为主

---

## 一、质量管理的内涵及基本内容

### 1.1 质量管理核心概念 ★★★★

| 概念 | 定义 |
|------|------|
| **质量** | 一组固有特性满足要求的程度 |
| **质量管理** | 在质量方面指挥和控制组织的协调的活动 |
| **质量方针** | 由组织最高管理者正式发布的关于质量方面的全部意图和方向 |
| **质量目标** | 在质量方面所追求的目的 |
| **质量策划** | 致力于制定质量目标并规定必要运行过程和相关资源以实现目标 |
| **质量控制** | 致力于满足质量要求 |
| **质量保证** | 致力于提供质量要求会得到满足的信任 |
| **质量改进** | 致力于增强满足质量要求的能力 |

> **速记**：质量管理四大活动 = 策划 + 控制 + 保证 + 改进

### 1.2 全面质量管理（TQM）★★★★

- **核心理念**：**三全一多样**——全员参与、全过程管理、全面质量、多种方法
- **关键原则**：
  - 以**顾客为中心**
  - **持续改进**
  - 全员参与
  - **过程方法**
  - 系统管理
  - 以事实为基础（**数据驱动**）
- **代表人物**：戴明（PDCA）、朱兰（质量三部曲）、克劳斯比（零缺陷）

### 1.3 质量管理发展阶段 ★★★

| 阶段 | 特征 |
|------|------|
| 质量检验阶段 | 事后把关，挑出不合格品 |
| 统计质量控制阶段 | 事前预防，运用统计方法 |
| 全面质量管理阶段 | 全员、全过程、全面质量 |

## 二、ISO 9000族标准与质量管理体系

### 2.1 ISO 9000族核心标准构成 ★★★★★

| 核心标准 | 内容 |
|----------|------|
| **ISO 9000** | 质量管理体系——**基础和术语** |
| **ISO 9001** | 质量管理体系——**要求**（唯一可用于认证的标准） |
| **ISO 9004** | 质量管理体系——**业绩改进指南** |
| **ISO 19011** | 管理体系审核**指南** |

> **必背**：ISO 9001是族标准中**唯一可用于第三方认证**的标准。

### 2.2 七项质量管理原则 ★★★★★

| 序号 | 原则 | 核心要点 |
|------|------|----------|
| 1 | **以顾客为关注焦点** | 理解顾客需求，超越顾客期望 |
| 2 | **领导作用** | 最高管理者确立统一方向 |
| 3 | **全员参与** | 各级人员是组织之本 |
| 4 | **过程方法** | 将活动作为过程管理，提高效率 |
| 5 | **改进** | 持续改进是组织永恒目标 |
| 6 | **循证决策** | 基于数据和信息的分析和评估做决策 |
| 7 | **关系管理** | 管理相关方关系以优化绩效 |

> **口诀**：顾领全过改循关（顾客领导全员过程改进循证关系）

### 2.3 PDCA循环（戴明环）★★★★★

| 阶段 | 含义 | 内容 |
|------|------|------|
| **P（Plan）** | 策划 | 制定目标、识别问题、制定计划 |
| **D（Do）** | 实施 | 执行计划、实施过程 |
| **C（Check）** | 检查 | 监测测量过程、报告结果 |
| **A（Act）** | 处置 | 采取改进措施、标准化成功经验 |

**PDCA特点**：
- **阶梯式上升**——每循环一次，质量提高一步
- **大环套小环**——各层级都有PDCA循环
- **持续不断**——循环往复、永不停歇

### 2.4 质量管理体系实施和评价 ★★★★

| 评价类型 | 执行主体 | 特点 |
|----------|----------|------|
| **内部审核** | 组织自身 | 自我检查，验证体系符合性和有效性 |
| **管理评审** | 最高管理者 | 评价体系适宜性、充分性、有效性 |
| **第二方审核** | 顾客/相关方 | 评价供方质量保证能力 |
| **第三方审核** | 认证机构 | 独立认证，颁发证书 |

> **易混淆点**：管理评审由**最高管理者**主持；内审由组织自行或委托进行。

### 2.5 常用质量管理工具 ★★★

| 工具 | 用途 |
|------|------|
| **排列图（帕累托图）** | 识别主要因素——"二八原则" |
| **因果图（鱼骨图）** | 分析问题的根本原因 |
| **直方图** | 展示质量数据的分布状态 |
| **控制图** | 监控过程是否处于受控状态 |
| **散布图** | 分析两个变量间的相关关系 |

## 三、《产品质量法》基本内容体系

### 3.1 产品质量法适用范围 ★★★★

- **产品**：经过加工、制作，用于销售的产品
- **不适用**：天然物品、建设工程、军工产品
- **监管体制**：**国家市场监督管理部门**主管全国产品质量监督工作

### 3.2 产品质量监督管理制度 ★★★★

| 制度 | 内容 |
|------|------|
| **产品质量抽查制度** | 对可能危及人身财产安全的产品进行抽查 |
| **质量认证制度** | 企业质量体系认证 + 产品质量认证 |
| **工业产品生产许可证制度** | 对重要工业产品实行生产许可证管理 |
| **标准化管理制度** | 强制性标准 + 推荐性标准 |
| **产品质量追溯制度** | 建立来源可查、去向可追、责任可究的追溯体系 |

### 3.3 生产者与销售者的产品质量义务 ★★★★

| 主体 | 主要义务 |
|------|----------|
| **生产者** | 产品不存在危及人身财产安全的不合理危险；符合国家标准；有产品质量检验合格证明；有中文标明的产品名称、厂名、厂址 |
| **销售者** | 进货检验义务；保持产品质量义务；不得销售失效变质产品 |

### 3.4 产品质量责任 ★★★

| 责任类型 | 说明 |
|----------|------|
| **民事责任** | 产品缺陷造成损害的赔偿责任（**无过错责任**） |
| **行政责任** | 责令改正、罚款、吊销营业执照等 |
| **刑事责任** | 构成犯罪的依法追究刑事责任 |

> **关键点**：生产者承担**无过错责任（严格责任）**；销售者承担**过错责任**（不能指明生产者时承担无过错责任）。

## 四、企业安全管理

### 4.1 安全生产法核心要点 ★★★★

| 要点 | 内容 |
|------|------|
| **方针** | **安全第一、预防为主、综合治理** |
| **主体责任** | 生产经营单位对本单位安全生产负**全面主体责任** |
| **主要负责人职责** | 建立/健全安全生产责任制；组织制定规章制度和操作规程；保证安全投入；组织安全培训；及时消除隐患；组织应急救援演练；及时如实报告事故 |
| **安全投入** | 生产经营单位应具备安全生产条件所必需的**资金投入** |
| **管理机构** | 从业人员超过**100人**的应设置安全生产管理机构或配备专职管理人员 |

### 4.2 安全生产管理关键制度 ★★★

| 制度 | 要点 |
|------|------|
| **安全生产责任制** | 最基本的制度，人人有责 |
| **安全教育培训制度** | 新员工"三级安全教育"（厂级、车间级、班组级） |
| **安全检查制度** | 定期检查和专项检查 |
| **隐患排查治理制度** | 发现隐患及时治理 |
| **事故报告和调查处理制度** | 事故发生后按规定及时上报 |
| **安全生产许可制度** | 矿山、危化品、建筑施工等高危行业须取得许可证 |

### 4.3 生产安全事故分级 ★★★

| 等级 | 死亡人数 | 重伤人数 | 直接经济损失 |
|------|----------|----------|-------------|
| **特别重大** | ≥30人 | ≥100人 | ≥1亿元 |
| **重大** | 10-29人 | 50-99人 | 5000万-1亿 |
| **较大** | 3-9人 | 10-49人 | 1000万-5000万 |
| **一般** | 1-2人 | 1-9人 | <1000万 |

> **速记**：3-10-30为死亡人数分界线；重伤为10-50-100；经济损失为1千万-5千万-1亿
