# 第一章 企业职能与战略决策

> **分值占比：约14%-15% | 案例分析高频命题点**
> **目标：60分万岁，抓大放小，以下标注出题概率**

---

## 第一节 企业与企业管理的职能

### 一、企业的概念与特征 ★★★

**核心定义**：企业是从事生产、流通、服务等经济活动，以产品或劳务满足社会需要，依法设立、自主经营、自负盈亏的经济组织。

**必背特征关键词**：
1. **经济性** -- 以营利为目的
2. **社会性** -- 满足社会需要
3. **独立性** -- 自主经营、自负盈亏
4. **法人性** -- 依法设立（公司制企业）

### 二、企业管理的职能 ★★★★

**必背六大职能（口诀：决组领控创激）**：

| 序号 | 职能 | 核心内容 | 记忆关键词 |
|------|------|----------|-----------|
| 1 | **决策** | 确定企业目标，选择最优方案 | "做正确的事" |
| 2 | **计划** | 制定行动方案，安排资源 | "蓝图与路径" |
| 3 | **组织** | 划分部门、明确权责、配置人员 | "搭班子" |
| 4 | **领导** | 指挥、协调、激励下属 | "带队伍" |
| 5 | **控制** | 监督执行、纠正偏差 | "纠偏" |
| 6 | **创新** | 推动技术和管理变革 | "变革图强" |

> **注意**：决策是首要职能，创新是推动企业发展的动力职能。

### 三、企业管理的任务 ★★★

1. **合理组织生产力** -- 优化资源配置
2. **维护和完善生产关系** -- 制度建设
3. **实现企业目标** -- 经济效益 + 社会效益

## 第二节 企业战略分析

### 一、企业战略的层次 ★★★★★

**必背三层架构（案例题常考判断属于哪层战略）**：

| 战略层次 | 别名 | 制定者 | 核心问题 | 关键词 |
|---------|------|-------|---------|--------|
| **总体战略（公司层战略）** | 公司战略 | 企业最高管理层 | "做什么业务" | 业务组合、资源配置方向 |
| **业务战略（业务层战略）** | 竞争战略 | 事业部/战略业务单位 | "怎么做这个业务" | 竞争优势、市场定位 |
| **职能战略** | 职能层战略 | 各职能部门 | "如何支撑业务战略" | 营销/财务/人力等具体策略 |

> **出题套路**：给你一段案例描述，问你属于哪个层次的战略。判断标准：涉及"做什么业务"=总体战略；涉及"如何竞争"=业务战略；涉及具体部门操作=职能战略。

### 二、企业战略的特征 ★★★★

| 特征 | 含义 |
|------|------|
| **全局性** | 从企业整体出发 |
| **长远性** | 着眼于未来 |
| **纲领性** | 是原则性和方向性的 |
| **抗争性/竞争性** | 针对竞争对手 |
| **相对稳定性** | 一定时期内保持不变 |

### 三、宏观环境分析 -- PEST分析 ★★★★

**PEST四要素速记表**：

| 字母 | 维度 | 典型分析内容 |
|------|------|------------|
| **P**（Political） | 政治法律环境 | 政策法规、政府行为、法律法规变化 |
| **E**（Economic） | 经济环境 | GDP、利率、汇率、通胀率、经济周期 |
| **S**（Social） | 社会文化环境 | 人口结构、消费观念、教育水平、价值观 |
| **T**（Technological） | 技术环境 | 新技术、技术转移速度、研发投入 |

### 四、行业环境分析 -- 波特五力模型 ★★★★★

**必考！五个竞争力量（画图理解）**：

```
        新进入者的威胁（潜在竞争者）
              |
  供应商的议价能力 --- 行业内现有企业竞争 --- 买方的议价能力
              |
        替代品的威胁
```

**五力详解**：

| 竞争力量          | 含义              | 影响因素（强度判断）                |
| ------------- | --------------- | ------------------------- |
| **行业内现有企业竞争** | 同行业企业之间的竞争      | 竞争者数量、行业增长率、产品差异化程度、退出壁垒  |
| **新进入者的威胁**   | 潜在竞争者进入的可能性     | 进入壁垒（规模经济、资金要求、品牌忠诚、政策限制） |
| **替代品的威胁**    | 替代产品对本行业的冲击     | 替代品的性价比、转换成本、客户对替代品的倾向    |
| **供应商的议价能力**  | 供应商提高价格或降低质量的能力 | 供应商集中度、替代投入品、前向一体化可能性     |
| **买方的议价能力**   | 客户压价或要求提高质量的能力  | 买方集中度、产品标准化程度、后向一体化可能性    |

> **记忆口诀**：行业内竞争+四个"来自外部"的力 = "一个中心四个基本点"

### 五、内部环境分析 ★★★★

| 分析工具 | 核心内容 | 关键词 |
|---------|---------|--------|
| **企业核心竞争力分析** | 不可模仿、不可替代的能力 | VRIN框架（有价值、稀有、不可模仿、不可替代） |
| **价值链分析** | 基本活动+辅助活动 | 基本活动五项：入库物流、生产运营、出库物流、营销销售、售后服务 |
| **波士顿矩阵** | 市场增长率 × 相对市场份额 | 明星、金牛、问题（问号）、瘦狗 |
| **内部因素评价矩阵（IFE）** | 评估企业内部优劣势 | 加权评分 |

**波士顿矩阵速记** ★★★★★：

| 象限 | 市场增长率 | 相对市场份额 | 策略 | 关键词 |
|------|----------|------------|------|--------|
| **明星** | 高 | 高 | 加大投资 | "烧钱但值" |
| **金牛** | 低 | 高 | 维持、收割 | "摇钱树" |
| **问题（问号）** | 高 | 低 | 选择性投资或放弃 | "鸡肋or明日之星" |
| **瘦狗** | 低 | 低 | 退出/清算 | "弃子" |

### 六、SWOT综合分析 ★★★★★

**必考！四象限战略组合**：

| | 优势（S） | 劣势（W） |
|------|----------|----------|
| **机会（O）** | **SO战略（增长型）**：发挥优势、利用机会 | **WO战略（扭转型）**：利用机会、克服劣势 |
| **威胁（T）** | **ST战略（多元化/防御型）**：利用优势、回避威胁 | **WT战略（防御型）**：减少劣势、回避威胁 |

> **记忆**：SO=最强（进攻），WT=最弱（防守），WO=补短板，ST=扬长避短

## 第三节 企业战略制定

### 一、企业总体战略 ★★★★★

**必考！总体战略四大类型对比表**：

| 战略类型 | 核心特征 | 适用条件 | 具体形式 | 典型案例 |
|---------|---------|---------|---------|---------|
| **发展战略（成长战略）** | 扩大规模、提高市场占有率 | 企业处于优势/市场有增长潜力 | 详见下表 | 华为全球化扩张 |
| **稳定战略** | 维持现状、不改变经营方向 | 市场环境稳定、企业效益良好 | 无变化战略、维持利润战略、暂停战略 | 成熟期企业维持 |
| **紧缩战略（收缩战略）** | 缩减规模、退出部分领域 | 经营困难/市场萎缩 | 转向战略、放弃战略、清算战略 | 诺基亚退出手机 |
| **组合战略** | 同时采用以上多种战略 | 大型企业多元化经营 | 同时在不同业务线采用不同战略 | 阿里不同业务线 |

### 二、发展战略的具体类型 ★★★★★

| 类型 | 含义 | 细分 | 特点 |
|------|------|------|------|
| **密集型成长战略** | 在现有业务领域内深耕 | 市场渗透、市场开发、产品开发 | 安索夫矩阵（产品×市场） |
| **一体化战略** | 沿产业链延伸 | **纵向一体化**（前向/后向）+ **横向一体化**（同行业并购） | 控制产业链 |
| **多元化战略** | 进入新的业务领域 | **相关多元化**（技术/市场相关）+ **非相关多元化**（无关联） | 分散风险 |

**一体化战略判断（常考）** ★★★★★：

| 类型 | 方向 | 含义 | 举例 |
|------|------|------|------|
| **前向一体化** | 向下游（客户方向） | 收购/控制分销商、零售商 | 制造商收购连锁零售店 |
| **后向一体化** | 向上游（供应商方向） | 收购/控制供应商 | 汽车厂收购轮胎厂 |
| **横向一体化** | 同行业水平方向 | 收购/合并同行业竞争者 | 滴滴合并快的 |

> **记忆诀窍**："前向"=向消费者方向（前方是客户），"后向"=向原料方向（后方是供应商）

**安索夫矩阵** ★★★★：

| | 现有产品 | 新产品 |
|------|---------|--------|
| **现有市场** | **市场渗透**（加大力度卖现有产品） | **产品开发**（向现有市场推新产品） |
| **新市场** | **市场开发**（将现有产品卖到新市场） | **多元化**（新产品+新市场） |

### 三、竞争战略（业务层战略） ★★★★★

**波特三大基本竞争战略**：

| 竞争战略 | 核心逻辑 | 实现途径 | 适用条件 | 风险 |
|---------|---------|---------|---------|------|
| **成本领先战略** | 以低成本获得竞争优势 | 规模经济、经验曲线、严格的成本控制 | 价格敏感的市场、标准化产品 | 技术变革使成本优势消失 |
| **差异化战略** | 提供独特的产品/服务 | 品牌、技术、设计、客户服务 | 客户愿意为差异化支付溢价 | 差异化被模仿、溢价过高 |
| **集中化战略** | 聚焦某一细分市场 | 在特定细分市场实现成本领先或差异化 | 细分市场有特殊需求 | 细分市场需求消失、竞争者入侵 |

> **集中化战略**又分为：集中成本领先 + 集中差异化

## 第四节 企业商业模式

### 一、商业模式的概念 ★★★★

**核心定义**：商业模式是企业创造价值、传递价值和获取价值的基本逻辑。

**商业模式的构成要素（魏朱六要素模型）**：

| 要素 | 内容 | 地位 |
|------|------|------|
| **定位** | 企业满足客户需求的方式 | 起点 |
| **业务系统** | 企业内外利益相关者的交易结构 | **核心** |
| **盈利模式** | 企业如何获得收入、控制成本 | 关键 |
| **关键资源能力** | 支撑商业模式运行的核心资源 | 支撑 |
| **现金流结构** | 企业现金流入和流出的模式 | 运行 |
| **企业价值** | 商业模式的最终评判标准 | 结果 |

### 二、商业模式画布（Business Model Canvas） ★★★★

**必背九模块**：

| 模块 | 核心问题 |
|------|---------|
| 1. **客户细分**（Customer Segments） | 为谁服务？目标客户是谁？ |
| 2. **价值主张**（Value Propositions） | 为客户解决什么问题？提供什么价值？ |
| 3. **渠道通路**（Channels） | 如何接触客户？ |
| 4. **客户关系**（Customer Relationships） | 与客户建立什么关系？ |
| 5. **收入来源**（Revenue Streams） | 如何赚钱？ |
| 6. **核心资源**（Key Resources） | 需要什么关键资源？ |
| 7. **关键业务**（Key Activities） | 需要做什么关键活动？ |
| 8. **重要合作**（Key Partnerships） | 需要哪些合作伙伴？ |
| 9. **成本结构**（Cost Structure） | 主要成本有哪些？ |

> **分组记忆**：右侧（1-5）= 客户端 = 价值传递；左侧（6-9）= 企业端 = 价值创造；底部 = 财务（收入-成本=利润）

### 三、价值创造三阶段 ★★★★

```
价值发现 --> 价值匹配 --> 价值获取
（识别需求）  （资源配置）  （实现盈利）
```

**成功商业模式的特征**：
1. 能够**创造独特的价值**
2. 具有**整合性和系统性**
3. **难以模仿**

## 第五节 企业经营决策

### 一、经营决策的类型 ★★★★

| 分类标准 | 类型 | 说明 |
|---------|------|------|
| **决策层次** | 战略决策/战术决策/业务决策 | 高层/中层/基层 |
| **决策问题的重要程度** | 战略决策/战术决策 | 全局性/局部性 |
| **决策问题的重复程度** | 程序化决策/非程序化决策 | 常规/非常规 |
| **环境可控程度** | 确定型/风险型/不确定型 | 已知/概率已知/未知 |

### 二、定性决策方法 ★★★

| 方法 | 特点 | 适用场景 |
|------|------|---------|
| **头脑风暴法** | 畅所欲言、禁止批评、追求数量 | 创新性决策 |
| **德尔菲法** | 匿名、多轮反馈、专家意见 | 长期预测、重大决策 |
| **名义小组技术** | 独立思考+小组讨论+投票 | 意见整合 |

### 三、定量决策方法 ★★★★★

**必考！三大类型决策方法汇总**：

#### （一）确定型决策方法

**1. 盈亏平衡分析法（量本利分析法）** ★★★★★

**核心公式**：
```
盈亏平衡产量 Q* = F / (P - V)

其中：
F = 固定成本
P = 单位产品售价
V = 单位变动成本
P - V = 单位边际贡献

利润 = (P - V) x Q - F
```

**考试题型**：给出固定成本、单价、单位变动成本，求盈亏平衡产量；或给定目标利润，求目标销量。

```
目标销量 = (F + 目标利润) / (P - V)
```

**2. 线性规划法** ★★★
- 在约束条件下求解最优解（最大利润或最小成本）

#### （二）风险型决策方法

**1. 期望损益决策法** ★★★★★

**核心公式**：
```
期望损益值 E = sum(Pi x Vi)

其中：
Pi = 第i种自然状态的概率
Vi = 第i种自然状态下的损益值

决策规则：选择期望损益值最大的方案
```

**2. 决策树法** ★★★★★

**决策树结构**：
```
         [决策点] --方案1--> [状态结点] --概率P1--> 损益值V1
                                    --概率P2--> 损益值V2
                   --方案2--> [状态结点] --概率P1--> 损益值V3
                                    --概率P2--> 损益值V4
```

**计算步骤**：
1. 从右向左计算各状态结点的期望损益值
2. 比较各方案的期望损益值
3. 选择期望值最大的方案

**符号约定**：
- 方框 = 决策点（引出方案枝）
- 圆圈 = 状态结点（引出概率枝）
- 三角形/数值 = 损益值

#### （三）不确定型决策方法 ★★★★★

**五大原则对比（必考表格）**：

| 决策原则 | 决策思路 | 选择标准 | 乐观程度 |
|---------|---------|---------|---------|
| **乐观原则（大中取大法）** | 在各方案最好结果中选最好的 | max(各方案最大收益值) | 最乐观 |
| **悲观原则（小中取大法）** | 在各方案最差结果中选最好的 | max(各方案最小收益值) | 最保守 |
| **折中原则** | 加权平均乐观和悲观结果 | max[alpha x max + (1-alpha) x min] | 中间 |
| **后悔值原则（最小最大后悔值法）** | 选择最大后悔值最小的方案 | min(各方案最大后悔值) | 较保守 |
| **等概率原则** | 假设各状态等概率 | max(各方案等概率期望值) | 中性 |

**后悔值计算步骤（必会计算）** ★★★★★：

```
第一步：找出每种自然状态下所有方案中的最大损益值
第二步：后悔值 = 该状态最大损益值 - 本方案损益值
第三步：计算每个方案在各种状态下的最大后悔值
第四步：选择最大后悔值中最小的方案
```

**后悔值公式**：
```
后悔值 Rij = max(Vij) - Vij

其中：
max(Vij) = 第j种状态下所有方案中的最大损益值
Vij = 第i个方案在第j种状态下的损益值

决策：min(max(Ri))  即选择最大后悔值最小的方案
```

**折中原则公式**：
```
折中损益值 = alpha x 最大收益值 + (1 - alpha) x 最小收益值

其中 alpha 为乐观系数（0 < alpha < 1）
决策：选择折中损益值最大的方案
```

| SO战略 | 优势+机会=增长型战略 |
| WO战略 | 劣势+机会=扭转型战略 |
