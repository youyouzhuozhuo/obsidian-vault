# 第三章 市场营销管理（占比约10%，必须掌握）

---

## 一、市场营销管理过程 ★★★★★必考

**市场营销管理**是指企业为实现其目标，创造、建立并保持与目标市场之间的互利交换关系而进行的分析、计划、执行与控制的过程。

### 管理过程的四个步骤

| 步骤 | 内容 | 要点 |
|------|------|------|
| ①分析市场机会 | 发现和评价市场机会 | **环境分析**（宏观/微观）、**消费者行为分析**、**竞争者分析** |
| ②选择目标市场 | 市场细分→目标市场选择→市场定位 | 即**STP战略**（详见第三节） |
| ③制定营销组合 | 设计营销策略组合 | **4P/4C/4R**策略组合（详见第四节） |
| ④管理营销活动 | 执行与控制 | 营销计划执行、**年度计划控制**、**盈利能力控制**、**效率控制**、**战略控制** |

> **记忆口诀**：分析机会→选择市场→制定组合→管理活动（"析选组管"）

## 二、市场营销信息系统 ★★★★高频

### 信息系统的构成

**市场营销信息系统（MIS）** 由四个子系统构成：

| 子系统 | 功能 | 说明 |
|--------|------|------|
| **内部报告系统** | 日常经营数据收集 | 订单、销售额、价格、库存、应收账款等 |
| **营销情报系统** | 外部环境信息获取 | 竞争对手动态、行业趋势、政策变化等 |
| **营销调研系统** | 专项问题研究 | **问卷调研**、**焦点小组**、**实验法**等 |
| **营销分析系统** | 数据分析与决策支持 | 统计库、模型库、**数据挖掘** |

### 市场营销调研的方法

| 方法分类 | 具体方法 | 特点 |
|----------|----------|------|
| **观察法** | 直接观察、间接观察 | 客观真实，但无法了解动机 |
| **询问法** | 面谈、电话、邮寄、网上调查 | 最常用，灵活但可能有偏差 |
| **实验法** | 实验室实验、现场实验 | 科学严谨，**因果关系**推断 |

> **易考点**：区分三种调研方法的优缺点及适用场景。

## 三、目标市场战略（STP） ★★★★★必考

**STP战略**是市场营销的核心战略框架，即**市场细分（Segmentation）→目标市场选择（Targeting）→市场定位（Positioning）**。

### （一）市场细分 ★★★★★必考

**市场细分**是指企业按照一定标准将整体市场划分为若干个子市场的过程。

#### 消费者市场细分标准（★★★★★）

| 细分变量 | 具体内容 |
|----------|----------|
| **地理变量** | 国家、地区、城市规模、气候、地形等 |
| **人口变量** | 年龄、性别、收入、职业、教育、家庭生命周期等 |
| **心理变量** | 生活方式、个性、价值观、社会阶层等 |
| **行为变量** | 购买动机、使用频率、**品牌忠诚度**、追求的利益等 |

#### 产业市场细分标准（★★★★）

| 细分变量 | 具体内容 |
|----------|----------|
| **地理变量** | 用户所在地 |
| **用户行业** | 最终用户的要求 |
| **用户规模** | 大客户、中客户、小客户 |
| **购买行为** | 采购方式、采购标准 |

> **考试重点**：能区分消费者市场与产业市场的细分标准差异。

#### 有效细分的条件（★★★★★）

有效市场细分必须满足以下条件（**五性**）：

1. **可衡量性（Measurability）**：子市场的规模和购买力可以衡量
2. **可进入性（Accessibility）**：企业能够进入并服务该市场
3. **可盈利性（Substantiality）**：子市场有足够的规模和利润空间
4. **差异性（Differentiability）**：子市场之间有明显的区别
5. **可行动性（Actionability）**：企业有资源和能力服务该市场

### （二）目标市场选择 ★★★★★必考

#### 三种目标市场策略

| 策略 | 含义 | 优点 | 缺点 | 适用条件 |
|------|------|------|------|----------|
| **无差异营销策略** | 以单一产品服务整个市场 | **规模经济**、成本较低 | 忽视差异，竞争激烈 | **同质市场**、需求差异小 |
| **差异性营销策略** | 以不同产品服务不同子市场 | 满足多元需求，扩大份额 | **成本高**、管理复杂 | **异质市场**、企业资源充足 |
| **集中性营销策略** | 集中资源服务一个或少数子市场 | **专业化**优势，精准营销 | **风险集中**，市场变化则受影响 | **中小企业**、资源有限 |

> **关键考点**：
> - 无差异→成本领先；差异性→市场覆盖；集中性→专业化
> - 选择策略需考虑：**企业资源**、**产品同质性**、**市场同质性**、**竞争者策略**、**产品生命周期**

### （三）市场定位 ★★★★★必考

**市场定位**是指企业根据竞争者现有产品在市场上所处的位置，针对顾客对该类产品某些特征或属性的重视程度，为本企业产品塑造与众不同的、给人印象鲜明的个性或形象。

#### 定位策略

| 策略 | 说明 |
|------|------|
| **避强定位** | 避开强竞争者，选择市场空隙；风险小，成功率较高 |
| **迎头定位** | 与强竞争者直接对抗；风险大，需有足够实力 |
| **重新定位** | 对原有产品进行二次定位；扭转不利局面 |

#### 定位方法

1. **产品差异化定位**：产品质量、特色、性能差异
2. **服务差异化定位**：附加服务、售后服务差异
3. **形象差异化定位**：品牌形象、企业文化差异
4. **人员差异化定位**：员工素质、服务态度差异

## 四、市场营销组合策略 ★★★★★必考

### （一）4P理论 ★★★★★必考

由**麦卡锡（McCarthy）** 提出，**科特勒（Kotler）** 完善。

| 4P要素 | 内容 | 关键点 |
|--------|------|--------|
| **产品（Product）** | 产品设计、品牌、包装、质量、服务、生命周期 | **产品整体概念**：核心产品→形式产品→期望产品→附加产品→潜在产品 |
| **价格（Price）** | 定价方法与策略 | **成本导向**、**需求导向**、**竞争导向** |
| **渠道（Place）** | 分销渠道设计与管理 | **渠道长度**（层级数）、**渠道宽度**（中间商数量） |
| **促销（Promotion）** | 广告、人员推销、营业推广、公共关系 | **推式策略**与**拉式策略** |

#### 产品生命周期各阶段营销策略（★★★★★）

| 阶段 | 特征 | 营销策略 |
|------|------|----------|
| **导入期** | 销售量低、成本高、利润为负 | 快速撇脂/缓慢撇脂/快速渗透/缓慢渗透 |
| **成长期** | 销量快速增长、利润上升 | 改进产品、开拓新市场、调整价格 |
| **成熟期** | 销量最大但增长放缓、竞争激烈 | 市场改良/产品改良/营销组合改良 |
| **衰退期** | 销量下降、利润减少 | 维持/集中/收缩/放弃 |

#### 定价方法（★★★★★）

| 定价方法 | 具体方式 | 公式/要点 |
|----------|----------|-----------|
| **成本导向** | 成本加成定价法 | 价格 = 单位成本 × (1 + 加成率) |
| | 目标利润定价法 | 价格 = (总成本 + 目标利润) / 预计销量 |
| | 盈亏平衡定价法 | 保本价格 = 固定成本 / 预计销量 + 单位变动成本 |
| **需求导向** | 理解价值定价法 | 以顾客对产品的认知价值定价 |
| | 需求差异定价法 | 不同顾客/时间/地点不同价格 |
| **竞争导向** | 随行就市定价法 | 以竞争者价格为基准 |
| | 密封投标定价法 | 以赢得投标为目标定价 |

#### 定价策略（★★★★）

| 策略 | 说明 |
|------|------|
| **新产品定价策略** | 撇脂定价（高价） / 渗透定价（低价） |
| **折扣与折让策略** | 现金折扣、数量折扣、功能折扣、季节折扣 |
| **心理定价策略** | 尾数定价、声望定价、参照定价 |
| **差别定价策略** | 顾客差别、产品形式差别、部位差别、时间差别 |

#### 分销渠道类型（★★★★）

| 分类标准 | 类型 |
|----------|------|
| **渠道长度** | **直接渠道**（零级）、**一级渠道**（生产者→零售商→消费者）、**二级渠道**、**三级渠道** |
| **渠道宽度** | **密集分销**（尽可能多）、**选择分销**（精选若干）、**独家分销**（一家） |

### （二）4C理论 ★★★★高频

由**劳特朋（Lauterborn）** 提出，从**消费者角度**出发。

| 4C要素 | 对应4P | 含义 |
|--------|--------|------|
| **顾客（Customer）** | Product | 先研究**顾客需求和欲望**，再生产产品 |
| **成本（Cost）** | Price | 顾客的**总成本**（货币成本+时间成本+精力成本+心理成本） |
| **便利（Convenience）** | Place | 为顾客提供**购买便利** |
| **沟通（Communication）** | Promotion | 与顾客**双向沟通**，而非单向促销 |

### （三）4R理论 ★★★★高频

由**唐·舒尔茨（Don Schultz）** 提出，强调**关系营销**。

| 4R要素 | 含义 |
|--------|------|
| **关联（Relevance）** | 与顾客建立**关联**，形成互助互求的关系 |
| **反应（Reaction）** | 对顾客需求**快速反应** |
| **关系（Relationship）** | 与顾客建立**长期稳定关系**，提高忠诚度 |
| **回报（Reward）** | 为企业带来**短期利润**和**长期价值** |

### （四）4P / 4C / 4R 对比 ★★★★★必考

| 对比维度 | 4P | 4C | 4R |
|----------|----|----|----|
| **提出者** | 麦卡锡 | 劳特朋 | 唐·舒尔茨 |
| **出发点** | 企业 | 消费者 | 关系/系统 |
| **核心理念** | 企业中心 | 顾客导向 | **关系营销** |
| **时代背景** | 卖方市场 | 买方市场 | 竞争全球化 |
| **关注重点** | 产品和交易 | 需求和满意 | 关系和回报 |
| **适用场景** | 传统营销 | 服务营销 | **现代关系营销** |

> **答题要点**：4P是基础框架（企业视角），4C是4P的补充（顾客视角），4R是升华（关系视角）。

## 五、市场营销新领域和新观念 ★★★★高频

### （一）绿色营销 ★★★★高频

**绿色营销**是指企业在营销活动中，以**环境保护**为出发点，通过开发绿色产品、制定绿色价格、建立绿色渠道、开展绿色促销，满足消费者的绿色消费需求。

- 核心：**可持续发展**
- 关键词：**绿色产品**、**绿色认证**、**环保标签**

### （二）网络营销 ★★★★高频

**网络营销**是指以互联网为媒介，实现企业营销目标的营销活动。

| 特点 | 说明 |
|------|------|
| **跨时空性** | 突破时间和空间限制 |
| **交互性** | 双向信息沟通 |
| **高效性** | 信息传递快速 |
| **经济性** | 降低营销成本 |
| **个性化** | 精准营销 |

主要方式：**搜索引擎营销**、**社交媒体营销**、**内容营销**、**电商营销**、**短视频营销**

### （三）体验营销 ★★★★高频

**体验营销（伯德·施密特）** 是指企业通过让目标顾客**体验产品或服务**，促使顾客认知、喜欢并购买的一种营销方式。

五大体验模块（**SEMM模型**）：

| 模块 | 内容 |
|------|------|
| **感官体验（Sense）** | 视觉、听觉、触觉等 |
| **情感体验（Feel）** | 激发顾客情感共鸣 |
| **思考体验（Think）** | 启发顾客思考和创新 |
| **行动体验（Act）** | 引导顾客参与互动 |
| **关联体验（Relate）** | 建立与社会群体的联系 |

### （四）关系营销 ★★★★高频

**关系营销**是指企业与顾客、供应商、分销商等建立**长期互利关系**的营销模式。

| 关系层次 | 说明 |
|----------|------|
| **一级关系营销** | 财务利益（价格折扣、赠送） |
| **二级关系营销** | 社会利益（建立社交联系） |
| **三级关系营销** | 结构性联系（提供增值服务） |

### （五）整合营销传播（IMC） ★★★★高频

**整合营销传播**是指将所有**传播工具和渠道**进行统一协调，向目标受众传递**一致的信息**。

- 核心思想：**一个声音，一个形象**
- 工具整合：广告、公关、促销、直复营销、人员推销、在线营销
- 关键：**以顾客为中心**，**统一信息**，**协调一致**

### （六）数字化营销与大数据营销 ★★★中频

| 概念 | 要点 |
|------|------|
| **数字化营销** | 利用数字技术和平台进行营销 |
| **大数据营销** | 基于**大数据分析**进行精准营销 |
| **精准营销** | 通过数据分析实现**个性化推送** |
| **私域流量** | 企业自主可控的流量池（如公众号、企业微信） |
| **用户画像** | 基于数据分析构建的用户标签体系 |

### （七）社会责任营销 ★★★中频

**社会责任营销**强调企业在营销活动中应考虑**社会整体利益**和**长期福利**。

- 核心：企业利润 + 消费者需求 + 社会利益（三方平衡）
- 体现：**诚信经营**、**消费者权益保护**、**环境保护**、**公益参与**

## 本章高频考点速记

### 必背要点清单

1. **STP战略**：市场细分→目标市场选择→市场定位
2. **市场细分五性**：可衡量、可进入、可盈利、差异性、可行动
3. **三种目标市场策略**：无差异（同质市场）、差异性（异质市场）、集中性（中小企业）
4. **三种定位策略**：避强定位、迎头定位、重新定位
5. **4P**：产品、价格、渠道、促销（企业视角）
6. **4C**：顾客、成本、便利、沟通（消费者视角）
7. **4R**：关联、反应、关系、回报（关系视角）
8. **产品生命周期四阶段**：导入期、成长期、成熟期、衰退期
9. **定价方法三导向**：成本导向、需求导向、竞争导向
10. **新产品定价策略**：撇脂定价（高价快收）、渗透定价（低价快占）
11. **分销渠道宽度**：密集分销、选择分销、独家分销
12. **关系营销三个层次**：财务利益→社会利益→结构性联系
13. **体验营销SEMM模型**：感官、情感、思考、行动、关联

### 案例分析答题框架

```
第一步：分析市场环境（宏观+微观）
第二步：确定STP战略
  - 按什么标准细分（地理/人口/心理/行为）
  - 选择哪种目标市场策略（无差异/差异性/集中性）
  - 定位策略是什么（避强/迎头/重新定位）
第三步：制定4P/4C/4R营销组合
第四步：结合新营销观念（数字化、体验、绿色、关系）
```
