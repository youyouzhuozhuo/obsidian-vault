## 一、财务管理目标和体系 ★★★★★必考

### （一）财务管理目标的三大观点

| 目标观点 | 含义 | 优点 | 缺点 |
|----------|------|------|------|
| **利润最大化** | 企业财务管理的最终目标是实现利润最大化 | ①计算简单，易于理解；②有利于资源合理配置；③有利于提高经济效益 | ①没有考虑**时间价值**；②没有考虑**利润与投入资本**的关系；③没有考虑**风险因素**；④容易导致**短期行为** |
| **股东财富最大化** | 企业通过合法经营，采取最优财务策略，使企业股东财富达到最大 | ①考虑了**风险因素**；②在一定程度上能**避免短期行为**；③对上市公司而言，**容易量化**（股票市值） | ①通常只适用于**上市公司**；②股价受多种因素影响，不能完全反映企业真实价值；③过于强调**股东利益**，忽视其他利益相关者 |
| **企业价值最大化** | 企业通过合理经营，采用最优财务政策，在考虑资金时间价值和风险的前提下，使企业总价值达到最大 | ①考虑了**时间价值**和**风险因素**；②兼顾各方**利益相关者**的利益；③更全面地反映企业**长期发展能力** | ①企业价值的**计量较为复杂**；②非上市企业的价值评估**难度较大** |

> **主流观点**：**企业价值最大化**是最优目标（兼顾股东、债权人、员工、社会利益）
> **记忆口诀**：利润最大化——"三没考虑"（时间、投入、风险）；股东财富——"三有缺陷"（仅上市、股价不稳、忽视他方）；企业价值——"三有优势但难计量"。

### （二）财务管理环节

| 环节 | 内容 |
|------|------|
| **财务预测** | 估计企业未来的融资需求 |
| **财务决策** | 财务管理的**核心环节**，包括筹资决策、投资决策、股利分配决策 |
| **财务预算** | 将财务决策的结果以货币形式具体化 |
| **财务控制** | 对财务活动的日常监督和调节 |
| **财务分析** | 对财务报表进行分析，评价企业财务状况 |
| **业绩评价** | 对企业经营成果进行综合评价 |

## 二、筹资管理 ★★★★★必考（计算重点）

### （一）筹资方式分类

| 分类标准 | 类别 | 说明 |
|----------|------|------|
| **按资金来源** | **权益筹资** | 吸收直接投资、发行股票、利用留存收益 |
| | **债务筹资** | 银行借款、发行债券、融资租赁、商业信用 |
| **按资金使用期限** | **短期筹资** | 期限在1年以内（商业信用、短期借款） |
| | **长期筹资** | 期限在1年以上（发行股票、债券、长期借款） |

### （二）个别资本成本计算 ★★★★★必考（必算）

#### 1. 长期借款资本成本

```
Kl = I × (1-T) / [L × (1-f)]

简化（不考虑筹资费用）：Kl = i × (1-T)
```

- **Kl**：长期借款资本成本
- **I**：年利息额
- **T**：企业所得税税率
- **L**：借款本金
- **f**：筹资费用率
- **i**：借款利率

> **核心原理**：债务利息可以**税前扣除**，因此有 (1-T) 的抵税效应。

#### 2. 债券资本成本

```
Kb = B × i × (1-T) / [B₀ × (1-f)]
```

- **Kb**：债券资本成本
- **B**：债券面值
- **i**：债券票面利率
- **T**：所得税税率
- **B₀**：债券发行价格（筹资额）
- **f**：筹资费用率

> **注意**：当债券**溢价发行**时 B₀ > B，资本成本低于票面利率的税后值；**折价发行**时 B₀ < B，资本成本高于票面利率的税后值。

#### 3. 普通股资本成本

**方法一：股利增长模型（戈登模型）**

```
Ks = D₁ / [P₀ × (1-f)] + g
```

- **Ks**：普通股资本成本
- **D₁**：预计第一年股利
- **P₀**：当前股票发行价格
- **f**：筹资费用率
- **g**：股利年增长率

**方法二：资本资产定价模型（CAPM）**

```
Ks = Rf + β × (Rm - Rf)
```

- **Rf**：无风险报酬率
- **β**：股票的贝塔系数（系统风险度量）
- **Rm**：市场平均报酬率
- **(Rm - Rf)**：市场风险溢价

#### 4. 留存收益资本成本

```
Kr = D₁ / P₀ + g
```

> 留存收益资本成本与普通股资本成本计算方法相同，但**没有筹资费用**。

**【计算例题】** 某公司长期借款200万元，年利率6%，筹资费用率0.5%，企业所得税税率25%；发行债券500万元，面值500万元，票面利率8%，筹资费用率2%；普通股300万元，预计第一年股利率10%，筹资费用率5%，股利增长率3%。计算各项个别资本成本。

```
解答：
- 长期借款资本成本：
  Kl = 200×6%×(1-25%) / [200×(1-0.5%)]
     = 9 / 199 = 4.52%

- 债券资本成本：
  Kb = 500×8%×(1-25%) / [500×(1-2%)]
     = 30 / 490 = 6.12%

- 普通股资本成本：
  Ks = 300×10% / [300×(1-5%)] + 3%
     = 30/285 + 3% = 10.53% + 3% = 13.53%
```

### （三）加权平均资本成本（WACC） ★★★★★必考

```
WACC = (E/V) × Re + (D/V) × Rd × (1-T)

简化记法：WACC = We × Ke + Wd × Kd × (1-T)
其中 We + Wd = 1
```

- **E**：权益资本的市场价值
- **D**：债务资本的市场价值
- **V = E + D**：总资本
- **Re**：权益资本成本
- **Rd**：债务税前资本成本
- **T**：企业所得税税率

> 权重可选择**账面价值**、**市场价值**或**目标价值**为基础确定。

**【计算例题】** 接上题，计算该公司的加权平均资本成本。

```
解答：
- 总资本 = 200 + 500 + 300 = 1000万元
- 权重：长期借款20%、债券50%、普通股30%

WACC = 20%×4.52% + 50%×6.12% + 30%×13.53%
     = 0.904% + 3.06% + 4.059%
     = 8.02%
```

### （四）杠杆效应 ★★★★★必考（计算重点）

#### 1. 经营杠杆系数（DOL）

**定义公式**：

```
DOL = (ΔEBIT / EBIT) / (ΔQ / Q)
```

**简化公式**（利用基期数据）：

```
DOL = (S - V) / (S - V - F) = M / EBIT = (EBIT + F) / EBIT
```

- **S**：销售额（收入）
- **V**：变动成本总额
- **F**：固定成本总额（不含利息费用）
- **M**：边际贡献 = S - V
- **EBIT**：息税前利润 = M - F

> **含义**：DOL越大，经营风险越高。固定成本F越大，DOL越大。

#### 2. 财务杠杆系数（DFL）

**定义公式**：

```
DFL = (ΔEPS / EPS) / (ΔEBIT / EBIT)
```

**简化公式**（利用基期数据）：

```
DFL = EBIT / (EBIT - I)
```

- **EPS**：每股收益
- **I**：利息费用（固定财务负担）
- **EBIT**：息税前利润

> **含义**：DFL越大，财务风险越高。利息I越大，DFL越大。当EBIT接近I时，DFL趋于无穷大。

#### 3. 总杠杆系数（DTL）

```
DTL = DOL × DFL = (ΔEPS / EPS) / (ΔQ / Q)
```

**简化公式**：

```
DTL = M / (EBIT - I) = (S - V) / (S - V - F - I)
```

> **含义**：DTL反映产销量变动对每股收益的放大效应。DTL = DOL × DFL 表明总杠杆是经营杠杆与财务杠杆的叠加效果。

**杠杆效应关系图**：

```
销售量变动(Q) ──[DOL放大]──> EBIT变动 ──[DFL放大]──> EPS变动
      |                                                ^
      |              DTL = DOL × DFL                   |
      +------------------------------------------------+
```

**【综合计算例题】** 某公司上年销售额S = 1000万元，变动成本率60%，固定成本总额F = 200万元，利息费用I = 60万元。计算DOL、DFL、DTL。

```
解答：
- 变动成本 V = 1000 × 60% = 600万元
- 边际贡献 M = 1000 - 600 = 400万元
- EBIT = 400 - 200 = 200万元

DOL = 400 / 200 = 2
→ 销售额增长1%，EBIT增长2%

DFL = 200 / (200 - 60) = 200/140 = 1.43
→ EBIT增长1%，EPS增长1.43%

DTL = DOL × DFL = 2 × 1.43 = 2.86
验证简化公式：DTL = 400/(200-60) = 400/140 = 2.86 ✓
→ 销售额增长1%，EPS增长2.86%
```

### （五）资本结构理论 ★★★★高频

#### 四大资本结构理论对比

| 理论 | 核心观点 | 企业价值表达式 | 最优资本结构 |
|------|----------|---------------|-------------|
| **无税MM理论** | 资本结构与企业价值**无关** | V_L = V_U | **不存在**最优资本结构 |
| **有税MM理论** | 负债产生利息**抵税效应**，负债越多企业价值越大 | V_L = V_U + PV(利息抵税) | 100%负债（理论极端） |
| **权衡理论** | 在负债的**税收利益**与**财务困境成本**之间权衡 | V_L = V_U + PV(利息抵税) - PV(财务困境成本) | **存在**最优资本结构 |
| **代理理论** | 在权衡理论基础上进一步考虑**代理成本** | V_L = V_U + PV(利息抵税) - PV(财务困境成本) - PV(代理成本) | **存在**最优资本结构 |
| **优序融资理论** | 信息不对称下企业融资有**优先顺序** | — | 融资顺序：内源→债务→股权 |

> **记忆口诀**：无税MM——"无关论"；有税MM——"负债越多越好"；权衡——"税收好处vs破产成本"；代理——"再加代理成本"；优序——"内部借钱发股"。

#### 最优资本结构的确定方法

| 方法 | 判断标准 |
|------|----------|
| **每股收益无差别点法** | 两种筹资方式下EPS相等时的EBIT，与预期EBIT比较 |
| **比较资本成本法** | 选择WACC最低的方案 |
| **公司价值分析法** | 选择使企业总价值最大的方案 |

**【每股收益无差别点例题】** 某公司当前资本结构为：普通股100万股，每股10元，无负债。现需追加筹资500万元，方案A：全部发行普通股50万股；方案B：全部发行债券，年利率8%。所得税率25%。

```
求EPS无差别点：

(EBIT - 0) × (1-25%) / 150 = (EBIT - 40) × (1-25%) / 100

EBIT / 150 = (EBIT - 40) / 100

100 × EBIT = 150 × (EBIT - 40)
100EBIT = 150EBIT - 6000
50EBIT = 6000
EBIT = 120万元

决策规则：
- 预期EBIT > 120万 → 选择方案B（债券筹资）
- 预期EBIT < 120万 → 选择方案A（股权筹资）
```

## 三、投资管理 ★★★★★必考（计算重点）

### （一）投资项目评价方法 ★★★★★必考

#### 1. 净现值法（NPV）

```
NPV = Σ [CFt / (1+r)^t] - C₀   （t从1到n）

其中：
  CFt = 第t年的净现金流量
  r   = 折现率（通常取WACC或要求的最低报酬率）
  n   = 项目寿命期
  C₀  = 初始投资额
```

**决策规则**：
- NPV > 0：项目**可行**，能增加企业价值
- NPV < 0：项目**不可行**
- NPV = 0：项目恰好达到要求回报

#### 2. 内部收益率法（IRR）

令 NPV = 0，求解 r：

```
Σ [CFt / (1+IRR)^t] = 0   （t从0到n）
```

**决策规则**：
- IRR > 要求的最低报酬率（基准收益率）：项目**可行**
- IRR < 要求的最低报酬率：项目**不可行**

**【插值法求解IRR】** 假设折现率 r₁ 时 NPV₁ > 0，折现率 r₂ 时 NPV₂ < 0：

```
IRR = r₁ + NPV₁ / (NPV₁ - NPV₂) × (r₂ - r₁)
```

> **注意**：r₂ - r₁ 一般不超过2%~5%以保证插值精度。

#### 3. 获利指数法（PI）

```
PI = Σ [CFt / (1+r)^t] / C₀ = NPV / C₀ + 1
```

**决策规则**：
- PI > 1：项目**可行**
- PI < 1：项目**不可行**
- PI = 1：NPV恰好为0

#### 4. 投资回收期法

**静态回收期**：

```
PP = 累计净现金流量首次≥0的年份 - 1 + |上年累计净现金流量| / 当年净现金流量
```

**动态回收期**：将各年净现金流量折现后再按同样方法计算。

#### 四种评价方法对比

| 方法 | 优点 | 缺点 | 考虑时间价值 |
|------|------|------|:---:|
| **NPV** | 考虑了全部现金流和时间价值；与股东财富最大化目标一致 | 不能反映投资效率 | 是 |
| **IRR** | 反映项目本身的收益率，易于理解 | 非常规现金流可能有多个IRR；互斥项目可能与NPV矛盾 | 是 |
| **PI** | 反映投资效率，便于资金有限时排序 | 不能反映投资规模 | 是 |
| **回收期** | 计算简单，反映流动性风险 | 忽略回收期后的现金流；静态法不考虑时间价值 | 否/是 |

> **口诀**："N大I高P过一"

### （二）综合计算例题

某公司拟投资一项目，初始投资100万元，项目寿命期5年，每年净现金流量如下：

| 年份 | 0 | 1 | 2 | 3 | 4 | 5 |
|------|---|---|---|---|---|---|
| 净现金流量（万元） | -100 | 20 | 30 | 30 | 40 | 50 |

折现率10%。

**（1）计算NPV**：

```
NPV = 20/(1.1)¹ + 30/(1.1)² + 30/(1.1)³ + 40/(1.1)⁴ + 50/(1.1)⁵ - 100
    = 18.18 + 24.79 + 22.54 + 27.32 + 31.05 - 100
    = 123.88 - 100 = 23.88万元

结论：NPV = 23.88 > 0，项目可行。
```

**（2）计算PI**：

```
PI = 123.88 / 100 = 1.24 > 1，项目可行。
```

**（3）计算静态回收期**：

| 年份 | 0 | 1 | 2 | 3 | 4 | 5 |
|------|---|---|---|---|---|---|
| 净现金流量 | -100 | 20 | 30 | 30 | 40 | 50 |
| 累计净现金流量 | -100 | -80 | -50 | -20 | 20 | 70 |

```
PP = 4 - 1 + |-20|/40 = 3 + 0.5 = 3.5年
```

## 四、成本费用管理 ★★★★高频

### （一）本量利分析（CVP分析） ★★★★★必考

#### 基本假设
1. 成本性态可分为**固定成本**和**变动成本**
2. 销售单价、单位变动成本、固定成本总额**保持不变**
3. 产销**基本平衡**
4. 只生产**一种产品**（或多产品组合不变）

#### 核心公式体系

| 指标 | 公式 | 说明 |
|------|------|------|
| **营业利润（P）** | P = (p - b) × Q - a | p=单价，b=单位变动成本，Q=销量，a=固定成本 |
| **单位边际贡献（m）** | m = p - b | 每销售一单位产品对利润的贡献 |
| **边际贡献总额（M）** | M = (p - b) × Q = S - V | S=销售收入，V=变动成本总额 |
| **边际贡献率（cmR）** | cmR = m/p = M/S | 边际贡献占收入的比率 |
| **变动成本率（bR）** | bR = b/p = V/S | 变动成本占收入的比率 |
| **关系** | **cmR + bR = 1** | 边际贡献率+变动成本率=1 |

#### 盈亏平衡点（保本点） ★★★★★

**单一产品**：

```
盈亏平衡点销量（Q*）= a / (p - b) = 固定成本 / 单位边际贡献

盈亏平衡点销售额（S*）= a / cmR = 固定成本 / 边际贡献率
```

**多产品**（加权平均法）：

```
综合盈亏平衡点销售额 = 固定成本总额 / 综合边际贡献率

综合边际贡献率 = Σ（某产品边际贡献率 × 该产品销售比重）
```

#### 安全边际

| 指标 | 公式 | 说明 |
|------|------|------|
| **安全边际量** | = Q(实际/预计) - Q* | 实际或预计销量超过盈亏平衡点销量的部分 |
| **安全边际额** | = S(实际/预计) - S* | 实际或预计销售额超过盈亏平衡点销售额的部分 |
| **安全边际率** | = 安全边际量 / 实际(预计)销量 | 比率越大，企业经营越安全 |

> **关键公式**：**利润 = 安全边际额 × 边际贡献率 = 安全边际量 × 单位边际贡献**

#### 安全程度判断标准

| 安全边际率 | 10%以下 | 10%~20% | 20%~30% | 30%~40% | 40%以上 |
|-----------|---------|---------|---------|---------|---------|
| 安全程度 | 危险 | 值得注意 | 比较安全 | 安全 | 很安全 |

**【计算例题】** 某企业生产A产品，单价50元，单位变动成本30元，固定成本总额80000元，正常销售量6000件。

```
要求：计算盈亏平衡点、安全边际、预期利润、边际贡献率。

解答：
- 单位边际贡献：m = 50 - 30 = 20元
- 边际贡献率：cmR = 20/50 = 40%
- 盈亏平衡点销量：Q* = 80000/20 = 4000件
- 盈亏平衡点销售额：S* = 80000/40% = 200000元
- 安全边际量：6000 - 4000 = 2000件
- 安全边际额：6000×50 - 200000 = 100000元
- 安全边际率：2000/6000 = 33.33%（安全）
- 预期利润：100000 × 40% = 40000元

验证：P = (50-30)×6000 - 80000 = 120000 - 80000 = 40000元 ✓
```

## 五、财务分析 ★★★★★必考

### （一）财务分析方法

| 方法 | 说明 |
|------|------|
| **比率分析法** | 最基本、最重要的分析方法 |
| **比较分析法** | 与历史、同行、预算比较 |
| **趋势分析法** | 分析多期数据的变化趋势 |
| **因素分析法** | 连环替代法、差额分析法 |

### （二）财务比率指标体系 ★★★★★必考（必须熟记所有公式）

#### 1. 偿债能力指标

| 类别 | 指标 | 公式 | 标准参考 | 分析要点 |
|------|------|------|----------|---------|
| **短期偿债** | **流动比率** | 流动资产 / 流动负债 | 一般≥2 | 过高可能资金闲置 |
| | **速动比率** | (流动资产-存货) / 流动负债 | 一般≥1 | 剔除变现能力最差的存货 |
| | **现金比率** | (货币资金+交易性金融资产) / 流动负债 | ≥0.2 | 最保守的短期偿债指标 |
| **长期偿债** | **资产负债率** | 负债总额 / 资产总额 × 100% | 适度(40%~60%) | 过高财务风险大，过低杠杆不足 |
| | **产权比率** | 负债总额 / 所有者权益 × 100% | — | 反映财务结构稳健程度 |
| | **权益乘数** | 资产总额 / 所有者权益 | — | = 1 + 产权比率 = 1/(1-资产负债率) |
| | **利息保障倍数** | EBIT / 利息费用 | ≥3较好 | 反映偿付利息的能力 |

#### 2. 营运能力指标（周转能力）

| 指标 | 公式 | 周转天数 |
|------|------|----------|
| **应收账款周转率** | 营业收入 / 应收账款平均余额 | 365 / 应收账款周转率 |
| **存货周转率** | 营业成本 / 存货平均余额 | 365 / 存货周转率 |
| **流动资产周转率** | 营业收入 / 流动资产平均余额 | 365 / 流动资产周转率 |
| **总资产周转率** | 营业收入 / 总资产平均余额 | 365 / 总资产周转率 |

> **分析要点**：周转率越高，周转天数越短，说明资产利用效率越高，营运能力越强。
> **注意**：平均余额 = (期初余额 + 期末余额) / 2

#### 3. 盈利能力指标

| 指标 | 公式 | 说明 |
|------|------|------|
| **销售毛利率** | (营业收入-营业成本) / 营业收入 × 100% | 反映产品初始盈利能力 |
| **销售净利率** | 净利润 / 营业收入 × 100% | 反映收入转化为利润的能力 |
| **总资产收益率（ROA）** | 净利润 / 总资产平均余额 × 100% | 反映全部资产的获利能力 |
| **净资产收益率（ROE）** | 净利润 / 所有者权益平均余额 × 100% | 反映股东投资的回报率 |
| **每股收益（EPS）** | (净利润-优先股股利) / 发行在外普通股加权平均数 | 反映每股盈利水平 |

#### 4. 发展能力指标

| 指标 | 公式 | 说明 |
|------|------|------|
| **营业收入增长率** | (本期营收-上期营收) / 上期营收 × 100% | 反映市场扩张能力 |
| **总资产增长率** | (期末总资产-期初总资产) / 期初总资产 × 100% | 反映规模增长 |
| **净利润增长率** | (本期净利润-上期净利润) / 上期净利润 × 100% | 反映盈利增长 |
| **资本积累率** | (期末所有者权益-期初所有者权益) / 期初所有者权益 × 100% | 反映资本保值增值 |

> **口诀**："偿营盈发"（偿债→营运→盈利→发展）

### （三）杜邦分析法 ★★★★★必考

杜邦分析法的核心是将**净资产收益率（ROE）**逐层分解：

```
ROE = 净利润 / 所有者权益
```

**第一层分解**：

```
ROE = ROA × 权益乘数
```

**第二层分解**：

```
ROE = 销售净利率 × 总资产周转率 × 权益乘数

即：ROE = (净利润/营业收入) × (营业收入/总资产) × (总资产/所有者权益)
```

**杜邦分析体系图**：

```
                      ROE（净资产收益率）
                     /                  \
            ROA（总资产收益率）    ×    权益乘数
           /          \                    |
    销售净利率  ×  总资产周转率      资产负债率
      /    \          /      \             |
  净利润  营业收入  营业收入  总资产    负债/资产
   /  \     |        |       / \
  收入-成本-税  收入构成   资产结构  负债+权益
```

> **三大驱动因素解读**：
> - **销售净利率**——反映**盈利能力**（利润端）
> - **总资产周转率**——反映**营运能力**（效率端）
> - **权益乘数**——反映**财务杠杆**（杠杆端）

**【杜邦分析计算例题】** 某公司2025年度有关数据如下：营业收入5000万元，净利润500万元，总资产平均余额4000万元，所有者权益平均余额2000万元。

```
解答：
- 销售净利率 = 500/5000 = 10%
- 总资产周转率 = 5000/4000 = 1.25次
- 权益乘数 = 4000/2000 = 2
- ROE = 10% × 1.25 × 2 = 25%

验证：ROE = 净利润/所有者权益 = 500/2000 = 25% ✓
```

## 六、并购重组 ★★★★高频

### （一）并购的类型

#### 按并购双方行业关系分类

| 类型 | 含义 | 主要动因 | 典型例子 |
|------|------|----------|---------|
| **横向并购** | 同行业、同一市场竞争者之间的并购 | 获得**规模经济**，扩大市场份额，消除竞争 | 两家航空公司合并 |
| **纵向并购** | 同一产业链上下游企业之间的并购 | 获得**交易内部化**效应，降低交易成本，保障供应链 | 汽车制造商并购零部件供应商 |
| **混合并购** | 不相关行业企业之间的并购 | 实现**多元化经营**，分散风险，发挥财务协同 | 家电企业并购金融公司 |

#### 按并购意愿分类

| 类型 | 含义 |
|------|------|
| **善意并购（协议收购）** | 并购双方通过友好协商达成一致 |
| **敌意并购（恶意收购）** | 不经目标公司管理层同意，直接向股东发出收购要约 |

#### 按出资方式分类

| 类型 | 含义 |
|------|------|
| **现金购买式并购** | 以现金购买目标企业的资产或股份 |
| **股份交易式并购** | 以并购方股票换取目标企业股票 |
| **承担债务式并购** | 并购方承担目标企业全部债务作为对价 |

### （二）并购的动因

| 动因类别 | 具体内容 |
|----------|---------|
| **获得规模经济** | 通过横向并购扩大生产规模，降低单位成本 |
| **获得协同效应** | 1+1>2的效应（详见下文） |
| **实现战略重组** | 调整业务结构，进入新市场或新领域 |
| **获得特殊资产和渠道** | 获取品牌、专利、技术、销售渠道等 |
| **降低代理成本** | 通过并购改善公司治理 |
| **管理者个人动机** | 扩大企业规模，增加管理者个人影响力 |

### （三）协同效应 ★★★★

| 类型 | 内容 | 来源 |
|------|------|------|
| **经营协同** | 并购后经营效率提升 | 规模经济、优势互补、降低交易成本 |
| **管理协同** | 管理效率提升 | 管理资源优化配置、管理能力扩散 |
| **财务协同** | 财务效率提升 | 合理避税、融资成本降低、资金互补、EPS自展效应 |

### （四）并购支付方式

| 支付方式 | 优点 | 缺点 |
|----------|------|------|
| **现金支付** | 速度快、交易简单、目标股东获得确定性收益 | 并购方资金压力大、可能丧失其他投资机会 |
| **股票支付** | 减轻现金压力、目标股东可分享并购后增值 | 稀释原有股东控制权、程序较复杂 |
| **混合支付** | 灵活组合，平衡各方利益 | 结构复杂、谈判难度大 |
| **承担债务** | 不需要大量现金 | 承担目标企业的债务风险 |
| **杠杆收购（LBO）** | 用目标公司资产融资 | 风险高、负债重 |

### （五）企业重组的方式

| 方式 | 含义 |
|------|------|
| **资产剥离** | 将部分资产或业务出售给第三方 |
| **分立** | 将部分业务分离成立新的独立企业 |
| **分拆上市** | 将子公司独立上市 |
| **股份回购** | 公司出资购回自身发行在外的股份 |

| 公式名称 | 公式 |

| DFL | DFL = EBIT/(EBIT-I) |
| DTL | DTL = DOL×DFL = M/(EBIT-I) |

### 投资管理公式

| 公式名称 | 公式 |
|----------|------|
| NPV | NPV = Σ CFt/(1+r)^t - C₀ |
| IRR | Σ CFt/(1+IRR)^t = 0 |
| IRR插值法 | IRR = r₁ + NPV₁/(NPV₁-NPV₂) × (r₂-r₁) |
| PI | PI = [Σ CFt/(1+r)^t] / C₀ |

### 本量利分析公式

| 公式名称 | 公式 |
|----------|------|
| 边际贡献 | M = (p-b) × Q |
| 边际贡献率 | cmR = (p-b)/p |
| 盈亏平衡销量 | Q* = a/(p-b) |
| 盈亏平衡销售额 | S* = a/cmR |
| 安全边际量 | = Q - Q* |
| 安全边际率 | = 安全边际量 / Q |
| 利润 | = 安全边际额 × cmR |

### 财务分析公式

| 公式名称 | 公式 |
|----------|------|

## 八、本章考试策略

### 案例分析题出题方向

| 出题方向 | 概率 | 核心考查内容 |
|----------|------|-------------|
| **筹资决策综合题** | ★★★★★ | 计算WACC、各杠杆系数，结合资本结构理论给筹资建议 |
| **投资决策综合题** | ★★★★★ | 给出项目现金流，计算NPV/IRR/PI/回收期，做投资决策 |
| **本量利分析综合题** | ★★★ | 给定成本数据，计算盈亏平衡点、安全边际、目标利润销量 |
| **财务分析综合题** | ★★★★★ | 给出财务报表数据，计算各类财务比率，进行杜邦分析和综合评价 |
| **并购重组分析题** | ★★★★ | 分析并购类型、协同效应、支付方式选择 |

### 应试要点

1. **公式要背熟**：WACC、DOL/DFL/DTL、NPV、本量利公式、财务比率公式是计算基础
2. **计算要准确**：一步一步计算，注意中间结果保留合理精度
3. **结论要明确**：计算后必须给出明确的决策结论（可行/不可行、选A/选B等）
4. **理论要联系实际**：在案例分析中结合资本结构理论解释计算结果
5. **时间管理**：计算题耗时较长，先做把握大的小问，再攻克综合大问
