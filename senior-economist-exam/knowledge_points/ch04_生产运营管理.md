# 第四章 生产运营管理（占比约8%）

## 一、生产运营战略 ★★★★高频

### 生产运营战略的概念

**生产运营战略**是指在企业总体战略指导下，根据目标市场和产品特点，对生产运营系统进行总体规划和决策，以获取**竞争优势**的战略性活动。

### 生产运营战略的类型

| 战略类型 | 核心内容 | 适用场景 |
|----------|----------|----------|
| **基于成本的战略** | 以低成本为核心竞争力 | 大批量标准化生产 |
| **基于质量的战略** | 以高质量为核心竞争力 | 精密制造、高端产品 |
| **基于柔性的战略** | 以快速适应市场变化为核心 | 多品种小批量 |
| **基于交期的战略** | 以快速交付为核心竞争力 | 时效性要求高的行业 |
| **基于服务的战略** | 以优质服务为核心竞争力 | 服务业、定制化产品 |

### 生产运营战略决策内容

| 决策领域 | 内容 |
|----------|------|
| **产品/服务决策** | 产品品种、质量水平、产量 |
| **工艺技术决策** | 生产工艺选择、自动化程度 |
| **生产能力决策** | 生产规模、产能规划 |
| **生产组织决策** | 生产布局、组织方式 |
| **供应链决策** | 自制或外包、供应商管理 |

## 二、生产过程组织 ★★★★★必考

### （一）生产过程的空间组织 ★★★★★必考

生产过程的空间组织是指在企业内部合理布置各生产单位，确定最优的**工艺路线**和**物流方向**。

| 组织形式 | 含义 | 优点 | 缺点 | 适用 |
|----------|------|------|------|------|
| **工艺专业化**（工艺原则） | 按工艺阶段设置生产单位 | 设备利用率高、适应性强 | 运输路线长、生产周期长 | **单件小批** |
| **对象专业化**（对象原则） | 按产品（零件）设置生产单位 | 运输路线短、生产周期短 | 设备利用率低、适应性差 | **大批大量** |
| **混合形式** | 结合上述两种 | 兼具两者优点 | 管理复杂 | 中等批量 |

> **考试重点**：区分"工艺专业化"与"对象专业化"的优缺点对比。

### （二）生产过程的时间组织 ★★★★★必考

生产过程的时间组织是指劳动对象在工序之间的**移动方式**。

#### 三种移动方式

| 移动方式 | 含义 | 生产周期 | 适用场景 |
|----------|------|----------|----------|
| **顺序移动方式** | 一批制品全部完成上道工序后才转到下道工序 | **最长** | 批量小、工序时间短 |
| **平行移动方式** | 每件制品完成后立即转到下道工序 | **最短** | 大批量、工序时间协调 |
| **平行顺序移动方式** | 既有平行又有顺序 | **介于两者之间** | 一般情况 |

#### 生产周期计算公式（★★★★★必考）

**（1）顺序移动方式**

$$T_{顺} = n \sum_{i=1}^{m} t_i$$

- T顺：顺序移动方式下一批制品的生产周期
- n：制品批量
- m：工序数
- t_i：第i道工序的单件时间

**（2）平行移动方式**

$$T_{平} = \sum_{i=1}^{m} t_i + (n-1) \times t_{max}$$

- t_max：所有工序中**最长**的单件工序时间

**（3）平行顺序移动方式**

$$T_{平顺} = \sum_{i=1}^{m} t_i + (n-1) \times (\sum t_{较大} - \sum t_{较小})$$

其中：
- t_较大：相邻两道工序中**较长**的工序时间（峰值为较大值）
- t_较小：相邻两道工序中**较短**的工序时间（谷值为较小值）

> **快速判断规则**：
> - 看相邻两道工序，前短后长（升）→ 较小值 = 较短的（前一道）
> - 看相邻两道工序，前长后短（降）→ 较大值 = 较长的（前一道）
> - 简化公式：T平顺 = n × Σti - (n-1) × Σmin(t_i, t_{i+1})

> **典型例题**：已知n=4件，有4道工序，各工序单件时间分别为t1=10min, t2=5min, t3=15min, t4=10min，求三种移动方式的生产周期。
>
> 解答：
> - T顺 = 4 × (10+5+15+10) = 4 × 40 = **160 min**
> - T平 = (10+5+15+10) + (4-1) × 15 = 40 + 45 = **85 min**
> - T平顺：较大值之和 = 10+15+15 = 40（注意首尾处理），较小值之和 = 5+5 = 10
>   T平顺 = 40 + (4-1) × (40-10) = 40 + 90 = **130 min** （具体计算以教材方法为准）

## 三、生产运营计划 ★★★★★必考

### （一）生产运营计划的层次

| 层次 | 计划类型 | 计划期 | 特点 |
|------|----------|--------|------|
| **战略层** | 长期生产计划 | 3-5年 | 产品方向、产能规划 |
| **战术层** | **生产计划（年度）** | 1年 | 品种、产量、产值 |
| **执行层** | **生产作业计划** | 月/周/日 | 具体任务分配与进度安排 |

### （二）生产计划主要指标（★★★★★）

| 指标 | 含义 | 说明 |
|------|------|------|
| **产品品种指标** | 规定生产的产品品种数 | 反映企业满足市场需求的能力 |
| **产品质量指标** | 规定产品质量应达到的水平 | 合格率、一等品率等 |
| **产品产量指标** | 规定企业生产的合格产品数量 | 实物量表示 |
| **产品产值指标** | 以货币表示的产量 | **总产值**、**商品产值**、**增加值**、**净产值** |

#### 产值指标区别（★★★★★）

| 产值类型 | 计算范围 |
|----------|----------|
| **总产值** | 企业在一定时期内生产的以货币表示的全部产品价值（含在制品） |
| **商品产值** | 企业在一定时期内生产的可供销售的产品价值 |
| **增加值** | 企业在生产过程中新创造的价值（= 总产值 - 中间投入） |
| **净产值** | 增加值扣除固定资产折旧后的净额 |

### （三）期量标准 ★★★★★必考

**期量标准**（又称作业计划标准）是指为编制生产作业计划而规定的一系列有关**生产数量和时间**的标准数据。

| 生产类型 | 主要期量标准 |
|----------|--------------|
| **大量大批** | **节拍**、流水线工作指示图表、在制品定额 |
| **成批生产** | **批量**、**生产间隔期**、生产周期、提前期、在制品定额 |
| **单件小批** | 生产周期、提前期 |

#### 关键期量标准及其关系（★★★★★）

| 期量标准 | 定义 | 计算关系 |
|----------|------|----------|
| **批量** | 一次投入（产出）的相同制品数量 | 批量 = 生产间隔期 × 平均日产量 |
| **生产间隔期** | 相邻两批同种制品投入（产出）的时间间隔 | 生产间隔期 = 批量 / 平均日产量 |
| **生产周期** | 从原材料投入到成品出产所经历的时间 | 各工序时间之和（含等待、运输） |
| **提前期** | 产品在各工艺阶段出产（投入）比成品出产应提前的时间 | 投入提前期 = 出产提前期 + 本阶段生产周期 |
| **在制品定额** | 在一定技术组织条件下，各生产环节为保证生产正常进行所需占用的最低在制品数量 | — |

#### 经济批量计算（★★★★★必考计算题）

**经济批量（EOQ）** 是指使**设备调整费用**与**库存保管费用**之和最小的生产批量。

$$Q^* = \sqrt{\frac{2 \times N \times A}{C \times i}}$$

- Q\*：经济批量
- N：年产量（年计划产量）
- A：每次设备调整费用（一次准备结束费用）
- C：单位产品成本（单价）
- i：年库存保管费率（单位产品年度保管费/单位产品成本）

总费用公式：

$$F = \frac{N}{Q} \times A + \frac{Q}{2} \times C \times i$$

当 Q = Q\* 时，设备调整费用 = 库存保管费用，总费用最小。

> **典型例题**：某产品年计划产量N=10000件，每次设备调整费A=500元，单位产品成本C=100元，年库存保管费率i=20%，求经济批量。
>
> 解答：Q\* = √(2 × 10000 × 500 / (100 × 0.2)) = √(10000000 / 20) = √500000 ≈ **707件**

### （四）生产作业计划编制方法 ★★★★高频

| 方法 | 适用类型 | 要点 |
|------|----------|------|
| **在制品定额法** | **大量大批** | 从成品车间开始，按反工艺顺序逐次确定各车间投入量和出产量 |
| **提前期法（累计编号法）** | **成批生产** | 按提前期转化为提前量，用**累计编号**规定车间任务 |
| **生产周期法** | **单件小批** | 按订货合同交货期和各工艺阶段生产周期，反推各车间投入出产时间 |

#### 在制品定额法计算（★★★★★）

$$某车间出产量 = 后续车间投入量 + 本车间半成品外销量 + (车间之间库存半成品定额 - 期初预计半成品库存量)$$

$$某车间投入量 = 本车间出产量 + 本车间允许的废品量 + (本车间在制品定额 - 期初预计本车间在制品量)$$

#### 累计编号法计算（★★★★★）

$$某车间出产累计号数 = 成品出产累计号数 + 该车间出产提前期 \times 成品平均日产量$$

$$某车间投入累计号数 = 成品出产累计号数 + 该车间投入提前期 \times 成品平均日产量$$

$$某车间计划期应完成的出产量 = 计划期末出产累计号数 - 计划期初已出产累计号数$$

## 四、生产运营控制 ★★★★高频

### （一）生产进度控制 ★★★★高频

| 控制内容 | 说明 |
|----------|------|
| **投入进度控制** | 控制产品投入的日期、数量、品种是否符合计划 |
| **工序进度控制** | 控制产品在各个工艺阶段的进度 |
| **出产进度控制** | 控制产品的出产日期、出产数量、出产成套性 |

### （二）在制品控制 ★★★★高频

**在制品控制**是指对生产过程中各阶段的在制品进行组织和控制。

| 控制内容 | 说明 |
|----------|------|
| **工艺在制品** | 正在各工序上加工的在制品 |
| **运输在制品** | 各工序之间正在运输的在制品 |
| **周转在制品** | 因工序间生产率不平衡而形成的在制品 |
| **保险在制品** | 为防止意外而设置的在制品储备 |

### （三）生产调度 ★★★中频

**生产调度**是生产作业控制的中心环节。

调度工作原则：
1. **计划性**：以计划为依据
2. **预见性**：防患于未然
3. **集中性**：统一指挥
4. **及时性**：迅速解决问题
5. **群众性**：依靠基层员工

### （四）库存控制 ★★★★高频

#### 库存分类

| 分类 | 类型 | 说明 |
|------|------|------|
| 按作用分 | **周转库存** | 为满足日常生产需要 |
| | **安全库存** | 为防止需求波动 |
| | **季节性库存** | 为满足季节性需求 |
| | **在途库存** | 运输途中的库存 |
| 按状态分 | **原材料库存** | 待投入生产的物料 |
| | **在制品库存** | 生产过程中的半成品 |
| | **成品库存** | 待销售的产品 |

#### ABC分类法（★★★★★）

| 类别 | 占品种比例 | 占资金比例 | 管理方式 |
|------|------------|------------|----------|
| **A类** | 10%-20% | 70%-80% | **重点管理**，严格控制，经常检查 |
| **B类** | 20%-30% | 15%-20% | 一般管理，定期检查 |
| **C类** | 50%-70% | 5%-10% | 简单管理，大量采购 |

#### 经济订购批量（EOQ）公式（★★★★★必考计算题）

$$Q^* = \sqrt{\frac{2 \times D \times S}{H}}$$

- Q\*：经济订购批量
- D：年需求量
- S：每次订购费用
- H：单位产品年储存费用

> **注意**：与生产经济批量公式的区别——生产批量公式中分母用C×i（产品成本×保管费率），库存订购批量公式中分母直接用H（单位年储存费）。

#### 订货点法（★★★★）

$$订货点 = 平均日需求量 \times 订货提前期 + 安全库存$$

## 五、现代生产管理方式 ★★★★★必考

### （一）精益生产（Lean Production） ★★★★★必考

**精益生产**起源于日本**丰田生产方式（TPS）**，由美国麻省理工学院在《改变世界的机器》一书中首次提出。

| 要素 | 内容 |
|------|------|
| **核心思想** | **消除一切浪费**，持续改进 |
| **基本目标** | 零库存、零缺陷、零浪费 |
| **两大支柱** | **准时化生产（JIT）** + **自动化（Jidoka）** |
| **基础** | **标准化**、**持续改进（Kaizen）**、**5S管理** |

#### 精益生产中的七种浪费（★★★★★）

| 序号 | 浪费类型 | 说明 |
|------|----------|------|
| 1 | **过量生产浪费** | 生产超过需求 |
| 2 | **等待浪费** | 等待材料、设备、信息 |
| 3 | **运输浪费** | 不必要的搬运 |
| 4 | **加工浪费** | 超出需要的加工 |
| 5 | **库存浪费** | 过多库存占用资金 |
| 6 | **动作浪费** | 不必要的操作动作 |
| 7 | **缺陷浪费** | 生产不合格品 |

> **记忆口诀**："产等运加库动缺"（产等运，加库动缺）

#### 精益生产的特征

1. **拉动式生产**：以客户需求为起点，后工序向前工序领取
2. **看板管理**：实现拉动式生产的工具
3. **团队工作**：多技能工人，团队协作
4. **持续改进**：不断发现问题、改善流程
5. **全面质量管理（TQM）**：质量贯穿全过程

### （二）准时制生产（JIT） ★★★★★必考

**准时制生产（Just-In-Time）** 是精益生产的核心，要求**在需要的时候，按需要的量，生产需要的产品**。

| 要素 | 内容 |
|------|------|
| **核心目标** | **零库存**，消除浪费 |
| **基本思想** | 只有当后工序需要时，前工序才生产 |
| **拉动方式** | **拉动式（Pull System）**，区别于传统的**推动式（Push System）** |
| **关键工具** | **看板（Kanban）** |

#### 看板的种类与功能（★★★★★）

| 类型 | 功能 |
|------|------|
| **取料看板** | 后工序向前工序取料的指令 |
| **生产看板** | 前工序进行生产的指令 |

#### 实施JIT的条件（★★★★★）

1. **生产均衡化**：生产负荷均匀
2. **生产同步化**：工序间衔接紧密
3. **看板管理**：信息传递工具
4. **全面质量管理**：保证产品质量
5. **多技能工人**：一人多岗
6. **与供应商建立长期合作关系**：准时供货

### （三）MRP / MRP II / ERP ★★★★★必考

#### MRP系统发展演进（★★★★★）

| 阶段 | 名称 | 覆盖范围 | 核心功能 |
|------|------|----------|----------|
| **MRP** | **物料需求计划**（Material Requirements Planning） | 物料管理 | 根据主生产计划计算**物料需求** |
| **MRP II** | **制造资源计划**（Manufacturing Resource Planning） | 整个制造系统 | 集成**物料+产能+财务** |
| **ERP** | **企业资源计划**（Enterprise Resource Planning） | 整个企业 | 集成**企业所有资源**（供应链、财务、人力等） |

> **注意**：MRP II不是MRP的简单升级，而是从**物料管理**扩展到**制造资源**的全面管理。

#### MRP的基本逻辑（★★★★★必考计算题）

MRP的三个基本输入：

| 输入 | 说明 |
|------|------|
| **主生产计划（MPS）** | 最终产品的生产数量和时间 |
| **物料清单（BOM）** | 产品结构树，记录组成产品的所有零部件 |
| **库存状态文件** | 当前库存量、在途量、已分配量等 |

MRP计算逻辑：

$$净需求 = 毛需求 - 预计可用库存 - 在途量$$

$$毛需求 = 独立需求 + 相关需求$$

- **独立需求**：来自市场的外部需求（成品）
- **相关需求**：由独立需求派生的内部需求（零部件，通过BOM展开计算）

#### MRP计算例题（★★★★★必考）

> **典型例题**：产品A由2个B和1个C组成，B由2个D组成。已知A的主生产计划为：第4周需100件，第7周需150件。B的当前库存为50件，C的当前库存为30件，D的当前库存为200件。各物料提前期：A=1周，B=2周，C=1周，D=2周。求各物料的计划订购量。

**解答思路**：
1. BOM展开：A需要B×2 + C×1；B需要D×2
2. 逐层计算：
   - A第4周需求100 → 第3周投入A（提前期1周）
   - B毛需求 = 100×2 = 200（第3周），现有库存50，净需求 = 200-50 = 150，提前期2周 → 第1周订购B 150件
   - C毛需求 = 100×1 = 100（第3周），现有库存30，净需求 = 100-30 = 70，提前期1周 → 第2周订购C 70件
   - D毛需求 = 150×2 = 300（第1周），现有库存200，净需求 = 300-200 = 100，提前期2周 → 提前期不足，需紧急处理
3. 同理处理第7周A的需求

#### ERP的核心模块（★★★★）

| 模块 | 功能 |
|------|------|
| **生产管理模块** | 生产计划、作业计划、生产控制 |
| **供应链管理模块** | 采购、库存、仓储、配送 |
| **财务管理模块** | 总账、应收应付、成本核算 |
| **人力资源管理模块** | 人事、薪酬、考勤、培训 |
| **销售管理模块** | 订单、客户管理、价格管理 |

### （四）大规模定制（Mass Customization） ★★★中频

**大规模定制**是指以**大批量生产的成本和效率**，提供满足**个性化需求**的产品或服务。

| 要素 | 内容 |
|------|------|
| **核心思想** | **低成本 + 个性化** |
| **实现途径** | **延迟差异化（延迟策略）** |
| **技术基础** | 柔性制造系统（FMS）、模块化设计、信息化管理 |

#### 大规模定制的四种类型（★★★★）

| 类型 | 说明 | 典型案例 |
|------|------|----------|
| **合作型定制** | 与客户共同设计 | 定制家具 |
| **透明型定制** | 企业根据客户数据自动定制 | 推荐算法 |
| **装饰型定制** | 标准产品+个性化包装 | 可口可乐定制瓶 |
| **适应性定制** | 标准产品可由客户自行调整 | 智能手机 |

#### 延迟策略（★★★★★）

**延迟策略（Postponement）** 是大规模定制的核心策略。

| 延迟类型 | 说明 |
|----------|------|
| **生产延迟** | 在生产过程中尽可能推迟差异化环节 |
| **物流延迟** | 在配送过程中根据客户需求进行差异化 |
| **延迟区分点** | 通用化生产与差异化生产的分界点 |

> **考试要点**：延迟策略的目的是在供应链中尽可能晚地进行产品差异化，以降低库存风险和满足个性化需求。

### （一）设备管理 ★★★中频

| 管理阶段 | 内容 |

| **设备更新** | 经济寿命计算、技术更新决策 |

#### 设备经济寿命计算（★★★★）

$$T^* = \sqrt{\frac{2 \times (P - L_n)}{G}}$$

- T\*：设备经济寿命（年）
- P：设备原值
- L_n：设备残值（简化计算时常设为0）
- G：设备年劣化增加值（每年增加的运行维护费用）

## 本章高频考点速记

### 必背要点清单

1. **空间组织三种形式**：工艺专业化（单件小批）、对象专业化（大批大量）、混合形式
2. **时间组织三种方式**：顺序移动（周期最长）、平行移动（周期最短）、平行顺序移动（介于两者）
3. **生产周期公式**：务必掌握三种移动方式的计算公式
4. **期量标准**：批量、生产间隔期、生产周期、提前期、在制品定额
5. **经济批量公式**：Q\* = √(2NA/Ci)
6. **作业计划编制三法**：在制品定额法（大量大批）、累计编号法（成批）、生产周期法（单件小批）
7. **ABC分类法**：A类（少品种高价值，重点管理）、B类、C类
8. **经济订购批量**：Q\* = √(2DS/H)
9. **精益生产**：核心是消除浪费，两大支柱是JIT和自动化
10. **七种浪费**：过量生产、等待、运输、加工、库存、动作、缺陷
11. **JIT**：拉动式生产、看板管理、零库存
12. **MRP→MRP II→ERP**：物料需求→制造资源→企业资源的演进
13. **MRP三输入**：主生产计划（MPS）、物料清单（BOM）、库存状态文件
14. **大规模定制**：核心策略是延迟差异化

### 计算题重点公式汇总

| 公式名称 | 公式 | 关键变量 |
|----------|------|----------|
| 顺序移动周期 | T = n × Σti | n=批量，ti=单件工序时间 |
| 平行移动周期 | T = Σti + (n-1) × t_max | t_max=最长工序时间 |
| 经济生产批量 | Q\* = √(2NA/Ci) | N=年产量，A=调整费，C=成本，i=保管费率 |
| 经济订购批量 | Q\* = √(2DS/H) | D=年需求，S=订购费，H=年储存费 |
| 设备经济寿命 | T\* = √(2(P-Ln)/G) | P=原值，Ln=残值，G=年劣化值 |
| 订货点 | R = d × L + SS | d=日均需求，L=提前期，SS=安全库存 |
