# 第六章 供应链管理

> 考试权重：**8%** | 优先级：中高 | 近3年考点重复率约60%，案例分析高频章节

---

## 一、供应链管理基本理论

### 1.1 供应链的概念 ★★★★

- **供应链**：围绕核心企业，通过对物流、信息流、资金流的控制，从采购原材料到制成品，最终由销售网络把产品送到消费者手中的将**供应商、制造商、分销商、零售商直到最终用户**连成一个整体的功能网链结构
- **核心思想**：**集成化**、**协同化**、**整体优化**

### 1.2 供应链类型 ★★★★

| 分类维度 | 类型 | 特点 |
|----------|------|------|
| 稳定性 | **稳定供应链** | 需求变化小，易于管理 |
| | **动态供应链** | 需求变化大，需不断调整 |
| 容量与需求 | **平衡供应链** | 供需匹配良好 |
| | **倾斜供应链** | 供需不匹配，供过于求或供不应求 |
| 功能模式 | **效率型供应链（ECR）** | 强调**低成本**、高效率，适合**功能型产品** |
| | **响应型供应链（QR）** | 强调**快速响应**，适合**创新型产品** |
| 发展趋势 | **敏捷供应链** | 灵活适应市场变化 |
| | **精益供应链** | 消除浪费，追求效率 |
| | **绿色供应链** | 兼顾环境友好与可持续发展 |
| | **数字供应链** | 基于数字技术的智能管理 |

> **必考对比**：效率型 vs 响应型供应链

| 维度 | 效率型供应链 | 响应型供应链 |
|------|-------------|-------------|
| **目标** | 低成本、高效率 | 快速响应市场需求 |
| **适用产品** | 功能型产品（需求稳定） | 创新型产品（需求不确定） |
| **库存策略** | 高周转、低库存 | 安全库存、缓冲库存 |
| **供应商选择** | 以**成本**为核心 | 以**速度/柔性**为核心 |
| **生产策略** | 按库存生产（MTS） | 按订单生产（MTO） |

### 1.3 牛鞭效应（Bullwhip Effect）★★★★★

**定义**：供应链中从下游到上游，需求波动**逐级放大**的现象，又称"需求放大效应"。

**四大成因**：

| 序号 | 成因 | 说明 |
|------|------|------|
| 1 | **需求预测修正** | 各节点企业独立预测，逐级放大 |
| 2 | **批量订购** | 为节约成本批量下单，造成需求波动 |
| 3 | **价格波动** | 促销打折导致提前购买，需求不稳定 |
| 4 | **限量供应与短期博弈** | 供不应求时虚报需求，加剧波动 |

**负面影响**：
- 库存积压（上游尤为严重）
- 生产能力浪费
- 服务水平下降
- 供应链总成本增加

**缓解措施**：
- **信息共享**（最根本）
- **缩短供应链**（减少中间环节）
- **减少批量订购**
- 建立**战略合作伙伴关系**
- 采用**VMI**（供应商管理库存）
- **延迟策略**（Postponement）

## 二、供应链环境下的物流管理

### 2.1 物流管理核心概念 ★★★

- **物流**：物品从供应地向接收地的实体流动过程，包括运输、仓储、包装、装卸搬运、流通加工、配送、信息处理等基本功能要素
- **物流管理目标**：在正确的时间、正确的地点、以正确的成本将正确的商品送达正确的顾客（"7R"原则）

### 2.2 第三方物流（3PL/TPL）★★★★

| 项目 | 内容 |
|------|------|
| **定义** | 由供方和需方以外的**第三方**物流企业提供物流服务的模式 |
| **本质** | **物流外包**，由专业公司承担物流功能 |
| **特点** | 专业化、信息化、**合同化**、规模化 |
| **优势** | 降低物流成本、聚焦核心业务、提高服务灵活性、专业技术服务 |
| **劣势** | 对第三方依赖、信息安全风险、控制力下降 |

**第三方物流与自营物流对比**：

| 维度 | 自营物流 | 第三方物流 |
|------|----------|-----------|
| 投资 | 大 | 小 |
| 专业性 | 较低 | 较高 |
| 灵活性 | 较低 | 较高 |
| 控制力 | 强 | 弱 |
| 适用企业 | 大型企业、物流需求稳定 | 中小企业、物流需求波动 |

### 2.3 冷链物流 ★★★★

| 项目 | 内容 |
|------|------|
| **定义** | 冷藏冷冻类物品在全程各环节始终处于**规定低温环境**下的物流体系 |
| **适用领域** | 生鲜食品、冷冻食品、医药、化工 |
| **关键要求** | **全程温控**、可追溯、设施设备专业化 |
| **核心特点** | 高投入、高技术要求、严格时效性 |
| **发展趋势** | 标准化、智能化、**绿色化** |

### 2.4 其他物流模式 ★★★

| 模式 | 特点 |
|------|------|
| **第一方物流** | 供应方（卖方）自行组织物流 |
| **第二方物流** | 需求方（买方）自行组织物流 |
| **第四方物流（4PL）** | **供应链集成商**，整合管理3PL资源，提供供应链解决方案 |

## 三、供应链的运作机制与绩效评价

### 3.1 供应链运作机制 ★★★★

| 机制 | 核心内容 |
|------|----------|
| **合作机制** | 上下游企业建立长期战略伙伴关系 |
| **决策机制** | 信息共享基础上协同决策 |
| **激励机制** | 利益分配与风险共担的激励体系 |
| **自律机制** | 企业自我约束与持续改进 |
| **风险防范机制** | 识别、评估、应对供应链风险 |

### 3.2 供应链协调控制模式 ★★★★

| 模式 | 特点 |
|------|------|
| **中心化控制** | 核心企业统一协调，集中决策 |
| **非中心化控制** | 各节点企业独立决策 |
| **混合式控制** | 关键环节集中控制 + 其他环节分散控制 |

### 3.3 供应链绩效评价方法 ★★★★

**BSC（平衡计分卡）在供应链中的应用**：

| 维度 | 关键指标 |
|------|----------|
| **财务** | 供应链总成本、利润率、资产利用率 |
| **客户** | 订单满足率、交货准时率、客户满意度、顾客保有率 |
| **内部流程** | 订单完成总周期、库存周转率、生产柔性 |
| **学习与成长** | 信息共享水平、合作稳定性、技术创新能力 |

**SCOR模型五大核心指标**：

| 指标 | 含义 |
|------|------|
| **可靠性** | 按时、按量、按质交付的能力 |
| **响应性** | 完成订单的速度 |
| **灵活性（敏捷性）** | 适应外部变化的能力 |
| **成本** | 供应链运营总成本 |
| **资产管理效率** | 资金和库存的使用效率 |

> **SCOR vs BSC**：SCOR是**流程导向**，关注供应链运营效率；BSC是**战略导向**，兼顾财务与非财务指标。

### 3.4 关键绩效指标速记 ★★★★

| 指标 | 计算方法/说明 |
|------|---------------|
| **订单满足率** | 按时按量交货的订单数 / 总订单数 |
| **完美订单率** | 完全无差错的订单比例 |
| **库存周转率** | 销售成本 / 平均库存 |
| **现金周转期** | 应收账款周转天数 + 库存周转天数 - 应付账款周转天数 |
| **供应链总成本** | 采购 + 运输 + 库存 + 管理 + 信息系统成本之和 |

## 四、供应链风险管理

### 4.1 供应链风险的概念与分类 ★★★★★

**定义**：供应链中各环节由于内外部不确定因素导致的偏离预期目标的可能性。

**按来源分类**：

| 类型 | 说明 | 举例 |
|------|------|------|
| **中断风险** | 供应链环节意外中断 | 自然灾害、政治动荡、供应商倒闭 |
| **延期风险** | 交付时间延迟 | 运输延误、生产故障 |
| **信息系统风险** | IT系统故障或数据安全 | 系统崩溃、数据泄露、网络攻击 |
| **预测风险** | 需求预测不准确 | 市场变化、信息失真 |
| **知识产权风险** | 核心技术或商业秘密泄露 | 信息共享中的泄密 |
| **采购风险** | 采购环节的不确定性 | 原材料价格波动、单一供应商依赖 |

### 4.2 供应链风险管理过程 ★★★★★

```
风险识别 → 风险评估 → 风险应对 → 风险监控
```

**（1）风险识别** ★★★★★

| 方法 | 说明 |
|------|------|
| **头脑风暴法** | 集体讨论，识别潜在风险 |
| **德尔菲法** | 专家匿名调查，多轮反馈 |
| **情景分析法** | 构建不同情景分析风险 |
| **流程分析法** | 沿供应链流程逐环节排查 |
| **SWOT分析** | 识别内外部优劣势和机会威胁 |

**（2）风险评估** ★★★★

| 方法 | 说明 |
|------|------|
| **风险矩阵法** | 按发生概率 × 影响程度构建矩阵 |
| **故障树分析（FTA）** | 从结果反推原因 |
| **蒙特卡洛模拟** | 随机模拟风险事件的影响 |
| **模糊综合评价法** | 处理不确定性因素的综合评价 |

**风险评估关键维度**：
- **发生概率**（高/中/低）
- **影响程度**（严重/一般/轻微）
- **风险等级** = 概率 x 影响

**（3）风险应对策略** ★★★★★

| 策略 | 内容 | 适用场景 |
|------|------|----------|
| **风险规避** | 改变计划消除风险 | 风险高且不可控 |
| **风险降低（减轻）** | 采取措施降低概率或影响 | 风险可控 |
| **风险转移** | 通过保险、外包等转移风险 | 可转移的财务风险 |
| **风险接受** | 不采取措施，承担后果 | 风险低且影响小 |

**具体应对措施**：

| 措施 | 说明 |
|------|------|
| **多元化供应商** | 避免单一供应商依赖（最常用） |
| **安全库存** | 设置缓冲库存应对需求波动 |
| **信息共享** | 提高供应链透明度 |
| **柔性设计** | 提高供应链弹性与敏捷性 |
| **合同管理** | 通过合同条款分担风险 |
| **应急预案** | 制定应急响应方案 |

**（4）风险监控** ★★★

- 建立风险预警指标体系
- 定期评估和更新风险清单
- 跟踪风险应对措施的执行效果

### 4.3 供应链韧性建设 ★★★★（2026年大纲新增关注点）

- **供应链韧性**：供应链在中断事件发生后能够快速恢复到正常运营状态的能力
- **韧性建设关键要素**：
  - **冗余**：备用供应商、安全库存
  - **柔性**：快速切换生产和供应来源
  - **敏捷**：快速感知和响应变化
  - **协作**：供应链上下游紧密合作
  - **可视化**：全链条信息透明

### 4.4 绿色供应链管理 ★★★（2026年大纲新增关注点）

| 要点 | 内容 |
|------|------|
| **核心理念** | 将**环境保护**融入供应链全过程 |
| **关键内容** | 绿色采购、绿色制造、绿色仓储、绿色运输、逆向物流 |
| **实施路径** | 供应链低碳化、循环经济应用、碳足迹追踪 |
| **评价标准** | 环境合规率、碳排放强度、资源回收利用率 |
