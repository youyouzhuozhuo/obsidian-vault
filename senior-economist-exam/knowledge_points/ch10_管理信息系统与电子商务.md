# 第十章 管理信息系统与电子商务

> **考试权重：6% | 优先级：较低（但近年数字化主题升温，不可忽视）**

## 一、企业相关主要信息技术

### 1.1 云计算 【考查概率：★★★】

| 维度 | 内容 |
|------|------|
| **定义** | 通过互联网按需提供计算资源（服务器、存储、数据库、网络、软件等）的技术模式 |
| **核心特征** | 按需自助服务、广泛的网络访问、资源池化、快速弹性伸缩、可计量的服务 |
| **部署模式** | **公有云**（开放给公众）、**私有云**（企业独享）、**混合云**（公私结合）、**社区云**（特定群体共享） |
| **服务模式** | **IaaS**（基础设施即服务，如阿里云ECS）、**PaaS**（平台即服务，如Google App Engine）、**SaaS**（软件即服务，如钉钉、Salesforce） |

**考试易混点：** 区分三种服务模式——IaaS租硬件、PaaS租平台、SaaS租软件。

### 1.2 大数据 【考查概率：★★★】

| 维度 | 内容 |
|------|------|
| **定义** | 规模巨大、类型复杂，无法用传统数据处理工具在合理时间内获取、管理和处理的数据集合 |
| **5V特征** | **Volume**（大量）、**Velocity**（高速）、**Variety**（多样）、**Value**（低价值密度/高价值）、**Veracity**（真实性） |
| **应用场景** | 精准营销、风险控制、供应链优化、用户画像、信用评估 |
| **关键技术** | Hadoop/Spark分布式计算、数据挖掘、机器学习算法、数据可视化 |

**记忆口诀：** 大数"5V"——量大速快样多值真。

### 1.3 人工智能（AI） 【考查概率：★★★】

| 维度 | 内容 |
|------|------|
| **定义** | 模拟、延伸和扩展人类智能的理论、方法、技术及应用系统 |
| **核心技术** | **机器学习**（含深度学习）、自然语言处理（NLP）、计算机视觉、智能机器人、专家系统 |
| **企业应用** | 智能客服、智能制造、自动驾驶、智能推荐系统、预测性维护、智能决策支持 |
| **发展阶段** | 弱人工智能（专用AI）→ 强人工智能（通用AI）→ 超人工智能 |

### 1.4 物联网（IoT） 【考查概率：★★☆】

| 维度 | 内容 |
|------|------|
| **定义** | 通过信息传感设备，按约定的协议，将任何物品与互联网连接，进行信息交换和通信的系统 |
| **三层架构** | **感知层**（传感器、RFID、条码）、**网络层**（传输网络）、**应用层**（行业应用） |
| **关键技术** | RFID射频识别、传感器网络、M2M通信、嵌入式系统 |
| **企业应用** | 智能制造、智慧物流、智能仓储、产品溯源、远程监控 |

**考试要点：** 物联网三层架构的对应功能——感知层"看"、网络层"传"、应用层"用"。

### 1.5 区块链 【考查概率：★★☆】

| 维度 | 内容 |
|------|------|
| **定义** | 一种去中心化的分布式账本技术，通过密码学保证数据不可篡改和不可伪造 |
| **核心特征** | **去中心化**、不可篡改、公开透明、可追溯、集体维护 |
| **关键技术** | 分布式账本、非对称加密、共识机制（PoW/PoS/DPoS）、智能合约 |
| **企业应用** | 供应链金融、产品溯源、数字票据、跨境支付、数字版权 |

**重要概念区分：**

| 概念 | 区别 |
|------|------|
| **公有链** | 任何人可参与（如比特币） |
| **联盟链** | 授权节点参与（如超级账本），企业应用主流 |
| **私有链** | 单一组织控制 |

## 二、管理信息系统开发与应用

### 2.1 管理信息系统（MIS）概述 【考查概率：★★★】

**定义：** 一个以人为主导，利用计算机硬件、软件、网络通信等技术，对信息进行收集、传输、加工、存储、更新和维护的系统。

**MIS的层次结构：**

| 管理层次 | 对应系统 | 功能 |
|----------|----------|------|
| **战略层**（高层） | 经理信息系统（EIS）/ 决策支持系统（DSS） | 长期规划、战略决策 |
| **战术层**（中层） | 管理信息系统（MIS） | 计划、控制、管理报告 |
| **作业层**（基层） | 事务处理系统（TPS） | 日常业务处理、数据采集 |

**MIS的开发方法比较：**

| 开发方法 | 特点 | 适用场景 | 优缺点 |
|----------|------|----------|--------|
| **结构化方法** | 自顶向下，分阶段开发 | 需求明确的大型系统 | 流程规范但周期长 |
| **原型法** | 快速构建原型，反复迭代 | 需求不明确的系统 | 快速反馈但可能不够严谨 |
| **面向对象法** | 以对象为中心，封装继承多态 | 复杂系统 | 复用性好但学习成本高 |
| **敏捷开发** | 迭代增量，快速交付 | 需求快速变化的系统 | 灵活但需团队协作能力强 |

### 2.2 企业资源计划（ERP） 【考查概率：★★★★】

| 维度 | 内容 |
|------|------|
| **定义** | 将企业所有资源进行整合集成管理，实现信息流、物流、资金流的高度统一和共享 |
| **核心思想** | **供应链管理**，将供应商、企业、客户整合为完整供应链 |
| **功能模块** | 财务管理、生产管理、采购管理、销售管理、库存管理、人力资源管理 |
| **发展阶段** | MRP（物料需求计划）→ MRP II（制造资源计划）→ **ERP**（企业资源计划）→ ERP II（协同商务） |

**ERP关键考点：**
- **MRP**：以物料为中心，解决"需要什么、何时需要、需要多少"
- **MRP II**：将生产、财务、销售、采购等集成，实现闭环管理
- **ERP**：在MRP II基础上扩展，覆盖企业全部资源，支持多工厂、多地点管理

### 2.3 客户关系管理（CRM） 【考查概率：★★★★】

| 维度 | 内容 |
|------|------|
| **定义** | 以客户为中心，通过对客户信息的收集、分析和利用，提升客户满意度和忠诚度的管理策略与系统 |
| **核心理念** | **以客户为中心**，而非以产品为中心 |
| **功能模块** | 客户信息管理、销售管理（销售漏斗）、营销管理、服务与支持、数据分析 |
| **关键指标** | 客户获取成本（CAC）、客户终身价值（CLV）、客户留存率、净推荐值（NPS） |

**CRM vs ERP区分：**
- **ERP** 面向企业内部资源整合，侧重"供应链"
- **CRM** 面向客户关系管理，侧重"客户价值"

### 2.4 供应链管理（SCM） 【考查概率：★★★】

| 维度 | 内容 |
|------|------|
| **定义** | 对从供应商到最终用户的整个供应链上的信息流、物流、资金流进行计划、协调、控制和优化的系统 |
| **核心目标** | **降低供应链总成本、提高服务水平、缩短响应时间** |
| **关键要素** | 供应商管理、采购管理、库存管理、物流配送、需求预测 |
| **发展趋势** | 数字化供应链、智能供应链、绿色供应链 |

### 2.5 信息系统开发步骤 【考查概率：★★☆】

| 阶段 | 主要工作 |
|------|----------|
| **系统规划** | 可行性分析、需求调研 |
| **系统分析** | 业务流程分析、数据流程图（DFD）、数据字典 |
| **系统设计** | 总体设计（模块结构）、详细设计（代码设计、数据库设计、输入输出设计） |
| **系统实施** | 编程、测试（单元测试→集成测试→系统测试→验收测试）、数据迁移、用户培训 |
| **系统维护** | 纠错性维护、适应性维护、完善性维护、预防性维护 |

## 三、电子商务运作系统

### 3.1 电子商务基本概念 【考查概率：★★★★】

**定义：** 以信息网络技术为手段，以商品交换为中心的商务活动。

**电子商务的"四流"：**

| 要素 | 说明 |
|------|------|
| **信息流** | 商品信息发布、广告、咨询等 |
| **资金流** | 在线支付、转账、结算 |
| **物流** | 商品配送、仓储、运输 |
| **商流** | 商品所有权的转移过程 |

**注意：** 四流中**信息流**是核心和纽带，贯穿始终。

### 3.2 电子商务运作体系 【考查概率：★★★】

**三大层次：**

| 层次 | 内容 |
|------|------|
| **网络层** | 网络基础设施（互联网、移动互联网、物联网等） |
| **平台层** | 电子商务平台（交易平台、支付平台、物流平台） |
| **应用层** | 具体商务应用（在线零售、在线服务、跨境贸易等） |

**关键支撑要素：** 电子支付体系、现代物流体系、信用体系、安全保障体系、法律法规体系。

### 3.3 电子商务的分类 【考查概率：★★★】

**按交易对象分类：**

| 类型 | 含义 | 典型案例 |
|------|------|----------|
| **完全电子商务** | 商流、资金流、物流、信息流均可在线完成 | 数字产品销售（电子书、软件、在线课程） |
| **非完全电子商务** | 不能完全依赖网络完成全部交易 | 实体商品电商（需要线下物流配送） |

**按交易主体分类：** 见下节电子商务模式。

## 四、电子商务的模式

### 4.1 电子商务模式对比 【考查概率：★★★★★】

| 模式 | 全称 | 含义 | 典型案例 | 特点 |
|------|------|------|----------|------|
| **B2B** | Business to Business | 企业对企业 | 阿里巴巴（1688）、慧聪网、中国制造网 | 交易金额大、决策流程长、客户数量少而稳定 |
| **B2C** | Business to Consumer | 企业对消费者 | 京东、天猫、亚马逊 | 标准化服务、用户量大、品牌导向 |
| **C2C** | Consumer to Consumer | 消费者对消费者 | 淘宝网、闲鱼、eBay | 进入门槛低、交易灵活、信任机制关键 |
| **O2O** | Online to Offline | 线上到线下 | 美团、大众点评、滴滴出行 | 线上引流、线下消费体验、本地化服务 |
| **C2B** | Consumer to Business | 消费者对企业 | 团购定制、海尔定制平台 | 反向定制、需求驱动、个性化 |
| **B2G** | Business to Government | 企业对政府 | 政府采购平台、电子政务 | 公开招标、合规性要求高 |

**考试高频考点：**
- **B2B** 是电子商务中交易额**最大**的模式
- **O2O** 的核心是将线下的商务机会带到线上，本质是**线上线下融合**
- 近年来出现**新零售**概念（马云提出），强调线上+线下+物流的深度融合

### 4.2 新兴电子商务模式 【考查概率：★★☆】

| 模式 | 说明 |
|------|------|
| **社交电商** | 基于社交网络传播商品信息（拼多多、小红书） |
| **直播电商** | 通过直播方式进行商品销售（抖音电商、淘宝直播） |
| **跨境电商** | 分属不同关境的交易主体（速卖通、SHEIN、Temu） |
| **社区团购** | 以社区为单位的团购模式（美团优选、多多买菜） |
| **新零售** | 线上线下+现代物流融合（盒马鲜生） |

## 五、网络营销

### 5.1 网络营销概述 【考查概率：★★★】

**定义：** 以互联网为主要手段，为达到一定营销目标而进行的营销活动。

**特点：** 跨时空性、交互性、多媒体性、经济性、技术性。

### 5.2 网络营销与传统营销对比 【考查概率：★★★】

| 对比维度 | 传统营销 | 网络营销 |
|----------|----------|----------|
| **传播媒介** | 电视、报纸、户外广告等 | 网站、社交媒体、搜索引擎等 |
| **互动性** | 单向传播 | 双向互动 |
| **成本** | 较高 | 相对较低 |
| **覆盖范围** | 受地域限制 | 全球覆盖 |
| **效果评估** | 难以精确量化 | 可精准追踪和量化 |
| **营销导向** | 4P（产品/价格/渠道/促销） | 4C（顾客/成本/便利/沟通） |

### 5.3 网络营销主要方法 【考查概率：★★★★】

| 营销方法 | 英文 | 说明 | 考试要点 |
|----------|------|------|----------|
| **搜索引擎营销** | SEM | 通过搜索引擎推广，包括SEO和付费搜索 | SEM包含SEO，SEO是免费的自然排名优化 |
| **搜索引擎优化** | SEO | 通过优化网站提升在搜索引擎中的自然排名 | 属于SEM的一部分，免费 |
| **电子邮件营销** | EDM | 通过电子邮件传递营销信息 | 成本低、精准但易被视为垃圾邮件 |
| **社交媒体营销** | SMM | 利用社交平台进行品牌传播和推广 | 强调互动和内容传播 |
| **内容营销** | Content Marketing | 通过创造有价值的内容吸引和保留受众 | 长期品牌建设，非直接销售 |
| **病毒式营销** | Viral Marketing | 通过用户的口碑传播实现信息快速扩散 | 利用社交网络效应，低成本高传播 |
| **联盟营销** | Affiliate Marketing | 通过合作伙伴推广并按效果付费 | 按CPS/CPA计费，风险低 |
| **大数据营销** | Data-driven Marketing | 基于用户数据进行精准定向营销 | 用户画像+精准投放 |

### 5.4 网络营销计费方式 【考查概率：★★★】

| 计费方式 | 英文 | 说明 |
|----------|------|------|
| **CPC** | Cost Per Click | 按点击付费（每次点击成本） |
| **CPM** | Cost Per Mille | 按千次展示付费 |
| **CPA** | Cost Per Action | 按行为付费（如注册、下载） |
| **CPS** | Cost Per Sale | 按销售付费（佣金模式） |
| **CPT** | Cost Per Time | 按时间付费（包月/包年） |

**记忆口诀：** CPC点击、CPM千展、CPA行动、CPS销售、CPT时间。

### 5.5 网络营销策略 【考查概率：★★★】

**4C理论（网络营销导向）：**

| 要素 | 英文 | 含义 |
|------|------|------|
| **顾客** | Consumer | 先了解顾客需求，再提供产品 |
| **成本** | Cost | 顾客获取和使用产品的总成本 |
| **便利** | Convenience | 方便顾客购买和使用 |
| **沟通** | Communication | 与顾客双向交流，建立关系 |

**与4P理论的区别：**
- **4P**（产品/价格/渠道/促销）以**企业**为中心
- **4C**（顾客/成本/便利/沟通）以**顾客**为中心

1. **优先掌握**：电子商务模式的识别与应用（结合案例分析）、三大管理系统（ERP/CRM/SCM）

3. **理解为主**：信息技术在企业中的应用场景（结合数字化转型趋势）
4. **关注热点**：AI赋能企业管理、新零售、社交电商、数据安全与隐私保护
