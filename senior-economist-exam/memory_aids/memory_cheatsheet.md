# 高频考点速记口诀

> 考前速查，每个知识点用最简短的方式记住。

---

## 第1章 企业战略

**总体战略四类型**：发展稳定紧缩混合
- **口诀**："发改稳缩混"

**竞争战略三选一**：成本领先、差异化、集中化
- **口诀**："成差集"

**SWOT四策略**：SO增长、WO扭转型、ST多元化、WT防御
- **口诀**："增扭多防"

**BCG矩阵**：明星（高增长高份额）、金牛（低增长高份额）、问题（高增长低份额）、瘦狗（低增长低份额）
- **口诀**："明金问狗"

**五力模型**：现有竞争者、潜在进入者、替代品、供应商议价、买方议价
- **口诀**："现进替供买"

---

## 第2章 公司治理

**三会一层**：股东会（权力机构）、董事会（决策机构）、监事会（监督机构）、管理层（执行机构）
- **口诀**："权决监执"

**董事义务**：忠实义务、勤勉义务
- **口诀**："忠勤"

**组织结构类型**：U型（职能制）、M型（事业部）、H型（控股）、矩阵制
- **口诀**："职事控矩"

---

## 第3章 市场营销

**STP**：Segmentation（细分）→ Targeting（目标）→ Positioning（定位）
- **口诀**："细分→选定→站住"

**4P**：Product（产品）、Price（价格）、Place（渠道）、Promotion（促销）
- **口诀**："品价渠促"

**4C**：Customer（顾客需求）、Cost（成本）、Convenience（便利）、Communication（沟通）
- **口诀**："需成便沟"

**产品生命周期**：导入→成长→成熟→衰退
- **口诀**："导成成熟衰"
- 导入期：快取/慢取策略
- 成长期：扩大市场、改善品质
- 成熟期：市场改良、产品改良、营销组合改良
- 衰退期：继续/集中/收缩/放弃

## 第7章 企业创新

**技术创新分类**：
- 产品创新 vs 工艺创新
- **口诀**："产新工新"

**创新模式**：自主创新、模仿创新、合作创新
- **口诀**："自模合"

**创新过程**：创意产生→创意筛选→概念开发→商业分析→产品开发→市场测试→商业化
- **口诀**："创筛概商产测商"（7步）

## 第8章 人力资源管理

**胜任力冰山模型**：知识技能（水面以上）→价值观→自我认知→特质→动机（水面以下）
- **口诀**："知技价自特动"

**绩效考核方法**：KPI（关键绩效）、OKR（目标关键结果）、BSC（平衡计分卡）、360度
- **口诀**："KOB三"

**BSC四个维度**：财务、客户、内部流程、学习与成长
- **口诀**："财客内学"

**薪酬类型**：岗位薪酬、技能薪酬、绩效薪酬
- **口诀**："岗技效"

## 第9章 财务管理

**财务目标三观点**：利润最大化、股东财富最大化、企业价值最大化
- **口诀**："利股企"

**杠杆三兄弟**：
- 经营杠杆（DOL）= 息税前利润变动率/销量变动率
- 财务杠杆（DFL）= 每股收益变动率/息税前利润变动率
- 总杠杆（DTL）= DOL × DFL
- **口诀**："经财总=经×财"

**资本结构三理论**：
- MM理论（无税：资本结构无关；有税：负债越多越好）
- 权衡理论（负债税盾收益与财务困境成本的权衡）
- 优序融资理论（内源融资→债务→股权）
- **口诀**："无关权衡优"（M权优）

**投资决策四指标**：
- NPV（净现值）≥0 → 可行
- IRR（内部收益率）≥资本成本 → 可行
- PI（获利指数）≥1 → 可行
- 回收期 ≤ 基准回收期 → 可行
- **口诀**："N大I高P过一"

**财务分析四能力**：
- 偿债能力（流动比、速动比、资产负债率）
- 营运能力（应收账款周转、存货周转、总资产周转）
- 盈利能力（ROE、ROA、毛利率、净利率）
- 发展能力（营收增长率、利润增长率、总资产增长率）
- **口诀**："偿营盈发"

## 通用答题口诀

**万能对策四件套**：
1. 完善制度/体制
2. 优化机制/流程
3. 加强监督/管理
4. 提升能力/水平

**万能主体三层**：
- 政府层面：政策引导、法规完善、监管加强
- 企业层面：战略调整、管理优化、创新驱动
- 社会层面：行业自律、公众参与、舆论监督

**万能时间三段**：
- 短期：应急处理、风险控制
- 中期：制度建设、能力提升
- 长期：战略转型、文化塑造
