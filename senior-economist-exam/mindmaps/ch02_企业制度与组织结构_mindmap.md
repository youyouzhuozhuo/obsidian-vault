```
┌─────────────────────────────────────────────────────────────────┐
│                        企业制度三种类型对比                       │
├───────────────┬─────────────────┬─────────────────┬──────────────┤
```

## ⚙️ 第二节 公司法人治理结构

### 法人治理结构组成 ★★★★★
```
       ┌─────────────┐
       │  股东(大)会  │ ←── 最高权力机构
       └──────┬──────┘
              │ 选举/更换
              ↓
       ┌─────────────┐
       │    董事会    │ ←── 决策机构（对股东会负责）
       └──────┬──────┘
              │ 聘任/解聘
              ↓
       ┌─────────────┐
       │    经理层    │ ←── 执行机构（对董事会负责）
       └──────┬──────┘
              │
       ┌─────────────┐
       │    监事会    │ ←── 监督机构（对股东会负责）
       └─────────────┘
```

### 股东会 vs 董事会 vs 经理层职权对比 ★★★★
```
┌──────────────┬──────────────────────────────────────────────┐
│    股东(大)会  │  决定经营方针和投资计划                       │
│              │  选举更换非职工代表董事、监事                   │
│              │  修改章程、增减资本、合并分立解散(2/3以上)      │
├──────────────┼──────────────────────────────────────────────┤
│    董事会     │  执行股东会决议                              │
│              │  决定经营计划和投资方案                        │
│              │  决定内部管理机构设置、聘任经理层              │
│              │  制定基本管理制度                            │
├──────────────┼──────────────────────────────────────────────┤
│    经理层     │  主持生产经营管理                            │
│              │  组织实施董事会决议                            │
│              │  拟定内部管理机构设置方案、基本管理制度        │
└──────────────┴──────────────────────────────────────────────┘
```

```
党委会前置研究讨论 ──→ 董事会决策 ──→ 经理层执行
```

## 🏗️ 第四节 企业组织结构

### 四种基本类型对比 ★★★★★
```
┌─────────────────────────────────────────────────────────────────┐
│                      组织结构四种类型对比                        │
├─────────┬─────────┬─────────┬─────────┬────────────────────────┤
│   类型   │   U型    │   M型    │   H型    │       矩阵制          │
│  (集权)  │ (事业部) │  (控股)  │  (双重领导)                     │
├─────────┼─────────┼─────────┼─────────┼────────────────────────┤
│ 权力特征 │ 高度集权 │ 集权+分权│ 高度分权│ 双重领导              │
│ 划分依据 │   职能   │ 产品/地区│   产权   │ 职能+项目             │
│ 法律地位 │ 无法人   │ 无法人   │ 子公司独立│ 无法人                │
│ 管理关系 │ 垂直指挥 │ 总部-事业部│ 母子产权│ 职能+项目经理双重    │
│ 首创者   │    —     │  斯隆(GM) │    —     │    —                  │
└─────────┴─────────┴─────────┴─────────┴────────────────────────┘
```

> 💡 记忆口诀：U=统一(集权)，M=多事业部(斯隆)，H=控股(产权)，矩阵=两条线交叉

### 组织变革方向 ★★★★★
```
┌─────────────────────────────────────────────────────────────┐
│                     "扁柔网合"                               │
├──────────────┬──────────────┬──────────────┬────────────────┤
│    扁平化     │    柔性化    │    网络化     │     合作化      │
│  减少管理层次 │ 增强适应性    │ 打破组织边界   │ 强化团队协作    │
│  缩短决策链   │ 动态调整      │ 跨组织合作     │ 打破部门墙      │
└──────────────┴──────────────┴──────────────┴────────────────┘
```

### 勒温三阶段变革模型 ★★★★
```
解冻期 ──→ 变革期 ──→ 再冻结期
(打破旧状态)  (实施变革)   (巩固新状态)
```
