## 📣 第四节 网络营销

### 计费方式 ★★★
```
┌─────────────────────────────────────────────────────────────┐
│                    网络营销计费方式                          │
├──────────────┬──────────────────────────────────────────────┤
│     方式       │                   含义                      │
├──────────────┼──────────────────────────────────────────────┤
│     CPC       │  Cost Per Click ── 按点击付费                │
│     CPM       │  Cost Per Mille ── 按千次展示付费            │
│     CPA       │  Cost Per Action ── 按行为付费               │
│     CPS       │  Cost Per Sale ── 按销售付费（佣金）          │
│     CPT       │  Cost Per Time ── 按时间付费（包月）         │
└──────────────┴──────────────────────────────────────────────┘

口诀：CPC点击、CPM千展、CPA行动、CPS销售、CPT时间
```
