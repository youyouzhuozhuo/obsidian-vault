### PDCA循环（戴明环）★★★★★
```
              ┌─────────┐
              │    P    │
              │  (策划)  │
              └────┬────┘
                   │
         ┌─────────┴─────────┐
         ↓                   ↓
    ┌─────────┐        ┌─────────┐
    │    A    │        │    C    │
    │  (处置) │   ←    │  (检查)  │
    └────┬────┘        └────┬────┘
         │                   │
         └─────────┬─────────┘
                   ↓
              ┌─────────┐
              │    D    │
              │  (实施)  │
              └────┬────┘
                   │
                   └────────→ (回到P，进入下一轮循环)
```

> 💡 特点：阶梯式上升、大环套小环、持续不断
