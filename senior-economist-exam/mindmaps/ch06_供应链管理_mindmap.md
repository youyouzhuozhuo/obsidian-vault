## 📊 第一节 供应链类型

### 效率型 vs 响应型供应链 ★★★
```
┌─────────────────────────────────────────────────────────────┐
│                效率型 vs 响应型供应链对比                     │
├─────────────────────┬───────────────────────────────────────┤
│      效率型(ECR)    │            响应型(QR)                   │
├─────────────────────┼───────────────────────────────────────┤
│  目标：低成本、高效率 │  目标：快速响应市场需求               │
│  适用：功能型产品    │  适用：创新型产品                      │
│  库存策略：高周转低库存│  库存策略：安全库存、缓冲库存          │
│  供应商选择：成本为核心│  供应商选择：速度/柔性为核心           │
│  生产策略：按库存生产  │  生产策略：按订单生产                  │
└─────────────────────┴───────────────────────────────────────┘
```

## 🐍 第二节 牛鞭效应

### 定义与成因 ★★★★★
```
┌─────────────────────────────────────────────────────────────┐
│                    牛鞭效应（需求放大效应）                   │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   零售商 ──→ 分销商 ──→ 制造商 ──→ 供应商                 │
│      ↑                                                 │
│      └────────────── 需求波动逐级放大 ──────────────────┘   │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                     四大成因                                 │
│  ① 需求预测修正 ── 各节点独立预测，逐级放大                  │
│  ② 批量订购   ── 批量下单造成需求波动                       │
│  ③ 价格波动   ── 促销打折导致提前购买                       │
│  ④ 限量供应   ── 供不应求时虚报需求                        │
├─────────────────────────────────────────────────────────────┤
│                     缓解措施                                 │
│  信息共享（最根本）│ 缩短供应链 │ VMI │ 延迟策略             │
└─────────────────────────────────────────────────────────────┘
```

## 📦 第三节 物流与库存

### 第三方物流 vs 自营物流 ★★★
```
┌─────────────────────────────────────────────────────────────┐
│              第三方物流(3PL) vs 自营物流                     │
├─────────────────────┬───────────────────────────────────────┤
│      第三方物流     │              自营物流                   │
├─────────────────────┼───────────────────────────────────────┤
│  投资：小          │  投资：大                              │
│  专业性：高         │  专业性：较低                          │
│  灵活性：高         │  灵活性：较低                          │
│  控制力：弱         │  控制力：强                            │
│  适用：中小企业     │  适用：大型企业、物流需求稳定           │
└─────────────────────┴───────────────────────────────────────┘
```

### 订货点法 ★★★
```
        订货点 = 平均日需求量 × 订货提前期 + 安全库存
```

## ⚡ 第五节 供应链风险管理

### 风险应对四大策略 ★★★★★
```
┌─────────────────────────────────────────────────────────────┐
│                    风险应对四大策略                          │
├──────────────┬──────────────────────────────────────────────┤
│   策略       │                   适用场景                   │
├──────────────┼──────────────────────────────────────────────┤
│  风险规避    │  风险高且不可控时，改变计划消除风险          │
│  风险降低    │  风险可控时，采取措施降低概率或影响          │
│  风险转移    │  可转移的财务风险，通过保险、外包转移         │
│  风险接受    │  风险低且影响小时，不采取措施承担后果         │
└──────────────┴──────────────────────────────────────────────┘
```
