## 📐 第一节 生产过程组织

### 空间组织两种形式 ★★★★★
```
┌─────────────────────────────────────────────────────────────┐
│                      空间组织形式对比                        │
├────────────────────────┬────────────────────────────────────┤
│     工艺专业化           │          对象专业化                 │
│     (工艺原则)          │          (对象原则)                 │
├────────────────────────┼────────────────────────────────────┤
│ 按工艺阶段设置生产单位    │ 按产品/零件设置生产单位            │
│ 设备利用率高             │ 运输路线短                         │
│ 适应性强                 │ 生产周期短                         │
│ 运输路线长               │ 设备利用率低                       │
│ 生产周期长               │ 适应性差                           │
├────────────────────────┼────────────────────────────────────┤
│     适用：单件小批       │       适用：大批大量                │
└────────────────────────┴────────────────────────────────────┘
```

### 时间组织三种移动方式 ★★★★★
```
顺序移动：一批制品全部完成上道工序后才转到下道工序
           (周期最长)

平行移动：每件制品完成后立即转到下道工序
           (周期最短)

平行顺序移动：既有平行又有顺序
              (周期介于两者之间)
```

#### 生产周期公式对比 ★★★★★
```
┌─────────────────────────────────────────────────────────────┐
│  T顺 = n × Σti                                            │
│      (n=批量, ti=各工序单件时间)                            │
│                                                             │
│  T平 = Σti + (n-1) × tmax                                  │
│      (tmax=最长工序时间)                                    │
│                                                             │
│  T平顺 = n×Σti - (n-1)×Σmin(ti, ti+1)                     │
└─────────────────────────────────────────────────────────────┘
```

## 📋 第二节 生产运营计划

### 计划层次 ★★★
```
┌─────────────────────────────────────────────────┐
│  战略层 ────── 长期生产计划（3-5年）            │
│       └── 产品方向、产能规划                     │
│                                                 │
│  战术层 ────── 年度生产计划                      │
│       └── 品种、产量、产值                       │
│                                                 │
│  执行层 ────── 生产作业计划（月/周/日）          │
│       └── 具体任务分配与进度安排                 │
└─────────────────────────────────────────────────┘
```

### 期量标准关系 ★★★★★
```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   批量 ×─────────────→ 生产间隔期                          │
│     ↑                        │                              │
│     │                        ↓                              │
│     │    ┌───────────────────────────────┐                  │
│     │    │  批量 = 生产间隔期 × 平均日产量│                  │
│     └──←─┘                               │                  │
│          └───────────────────────────────┘                  │
│                         ↓                                   │
│          ┌───────────────────────────────┐                  │
│          │    生产周期 + 提前期           │                  │
│          │  (从投入到成品的时间)         │                  │
│          └───────────────────────────────┘                  │
│                         ↓                                   │
│          ┌───────────────────────────────┐                  │
│          │      在制品定额               │                  │
│          │  (各环节需占用的最低在制品)    │                  │
│          └───────────────────────────────┘                  │
└─────────────────────────────────────────────────────────────┘
```

### 经济批量公式 ★★★★★
```
        ┌──────────────────────────────┐
        │   Q* = √(2 × N × A / C × i) │
        └──────────────────────────────┘
        
其中：N=年产量  A=每次调整费  C=单位成本  i=年保管费率
```
