## 📊 第一节 三大杠杆效应

### 经营杠杆系数（DOL）★★★★
```
┌─────────────────────────────────────────────────────────────┐
│              DOL = 边际贡献 / 息税前利润（EBIT）             │
│                 = (EBIT + F) / EBIT                       │
│                                                             │
│  含义：销售额变动1% → EBIT变动DOL%                         │
│  固定成本F越大 → DOL越大 → 经营风险越高                     │
└─────────────────────────────────────────────────────────────┘
```

### 财务杠杆系数（DFL）★★★★
```
┌─────────────────────────────────────────────────────────────┐
│              DFL = EBIT / (EBIT - I)                       │
│                                                             │
│  含义：EBIT变动1% → EPS变动DFL%                           │
│  利息I越大 → DFL越大 → 财务风险越高                        │
└─────────────────────────────────────────────────────────────┘
```

### 总杠杆系数（DTL）★★★★
```
┌─────────────────────────────────────────────────────────────┐
│              DTL = DOL × DFL                               │
│                 = 边际贡献 / (EBIT - I)                    │
│                                                             │
│  含义：销售额变动1% → EPS变动DTL%                          │
│  DTL = DOL × DFL：经营杠杆与财务杠杆的叠加效果             │
└─────────────────────────────────────────────────────────────┘
```

## 📈 第二节 投资决策

### NPV/IRR/PI对比 ★★★★★
```
┌─────────────────────────────────────────────────────────────┐
│                  投资决策三指标对比                           │
├──────────────┬───────────────┬──────────────────────────────┤
│    指标       │     公式       │           决策标准           │
├──────────────┼───────────────┼──────────────────────────────┤
│    NPV       │  ΣCFt/(1+r)^t │  NPV > 0 → 可行             │
│  (净现值)    │    - C₀       │                             │
├──────────────┼───────────────┼──────────────────────────────┤
│    IRR       │  ΣCFt/(1+IRR)^t│  IRR > 资本成本 → 可行     │
│ (内部收益率) │      = 0       │                             │
├──────────────┼───────────────┼──────────────────────────────┤
│    PI        │  [ΣCFt/(1+r)^t]│  PI > 1 → 可行            │
│  (获利指数)  │      / C₀      │                             │
└──────────────┴───────────────┴──────────────────────────────┘

口诀："N大I高P过一"
```

## 📉 第三节 本量利分析（CVP）

### 核心公式体系 ★★★★★
```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   边际贡献 M = (p - b) × Q                                │
│                    ↑                                       │
│            单位边际贡献 = 单价 - 单位变动成本                 │
│                                                             │
│   边际贡献率 cmR = (p - b) / p                            │
│                                                             │
│   营业利润 P = (p - b) × Q - a                            │
│                    ↑                    ↑                   │
│               边际贡献总额           固定成本               │
└─────────────────────────────────────────────────────────────┘
```

### 盈亏平衡点 ★★★★★
```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│         盈亏平衡销量 Q* = 固定成本 / 单位边际贡献            │
│                          = a / (p - b)                    │
│                                                             │
│         盈亏平衡销售额 S* = 固定成本 / 边际贡献率            │
│                          = a / cmR                        │
│                                                             │
│   安全边际量 = 实际销量 - 盈亏平衡销量                       │
│   安全边际率 = 安全边际量 / 实际销量                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 📋 第四节 财务分析

### 杜邦分析法 ★★★★★
```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│                    ROE = 销售净利率                         │
│                         × 总资产周转率                      │
│                         × 权益乘数                          │
│                                                             │
│         ┌─────────────────────────────────────────┐        │
│         │              净资产收益率(ROE)            │        │
│         └───────────────────┬─────────────────────┘        │
│                             │                               │
│              ┌──────────────┴──────────────┐              │
│              │         总资产收益率(ROA)     │              │
│              └──────────────┬──────────────┘              │
│                             │                               │
│              ┌──────────────┴──────────────┐              │
│              ↓                              ↓              │
│      ┌───────────────┐            ┌───────────────┐      │
│      │   销售净利率   │            │  总资产周转率  │      │
│      │ =净利润/营收  │            │ =营收/总资产  │      │
│      └───────────────┘            └───────────────┘      │
│                             │                               │
│                             ↓                               │
│                    ┌───────────────┐                       │
│                    │   权益乘数     │                       │
│                    │ =1/(1-资产负债率)│                   │
│                    └───────────────┘                       │
└─────────────────────────────────────────────────────────────┘
```

口诀："偿营盈发"
