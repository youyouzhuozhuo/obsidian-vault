```
        新进入者的威胁
              │

  议价能力       │       议价能力

        替代品的威胁
```

• 新进入者的威胁
• 替代品的威胁

• 买方议价能力

### 波士顿矩阵 ★★★★★
```
    高增长
      ↑
      │   明星          问题
      │  (烧钱但值)   (鸡肋or明日星)
      │
低份额 ├─────────────────┼──→ 高份额
      │   瘦狗          金牛
      │   (弃子)       (摇钱树)
      │
      └─────────────────────────→ 低增长
```

| 象限 | 特征 | 策略 |
|------|------|------|
| 明星 | 高增长+高份额 | 加大投资 |

| 瘦狗 | 低增长+低份额 | 退出清算 |

## 📋 第三节 企业战略制定

### 总体战略四大类型 ★★★★★
• 发展战略（成长战略）— 扩大规模、提高市占率
• 稳定战略 — 维持现状、不改变方向
• 紧缩战略（收缩战略）— 缩减规模、退出领域
• 组合战略 — 同时采用多种战略

### 一体化战略判断 ★★★★★
```
  ←── 供应端（后向一体化）     制造/生产     前向一体化（客户端）──→
       汽车厂收购轮胎厂          →           制造商收购连锁零售店

  ←───────────────────── 横向一体化（同行业） ──────────────────────→
       滴滴合并快的、优步中国合并
```

### 安索夫矩阵 ★★★★
```
                    现有产品              新产品
              ┌──────────────────────┬─────────────────────┐
    现有市场  │    市场渗透          │     产品开发        │
              │  (加大力度卖现有产品)  │  (向现有市场推新品)   │
              ├──────────────────────┼─────────────────────┤
      新市场  │    市场开发          │      多元化          │
              │  (将现有产品卖到新市场)│  (新产品+新市场)     │
              └──────────────────────┴─────────────────────┘
```

### 波特三大竞争战略 ★★★★★
• 成本领先战略 — 以低成本获得竞争优势
• 差异化战略 — 提供独特产品/服务
• 集中化战略 — 聚焦某一细分市场

## 📊 第五节 企业经营决策

### 不确定型决策五大原则 ★★★★★
```
┌────────────┬──────────────────┬─────────────────────────┐
│   原则     │     思路          │     选择标准             │
├────────────┼──────────────────┼─────────────────────────┤
│ 乐观原则   │ 在最好结果中选最好 │ max(各方案最大收益值)    │
│ 悲观原则   │ 在最差结果中选最好 │ max(各方案最小收益值)    │
│ 折中原则   │ 加权平均乐观悲观   │ max[α·max+(1-α)·min]   │
│ 后悔值原则 │ 最大后悔值中选最小 │ min(各方案最大后悔值)    │
│ 等概率原则 │ 假设各状态等概率   │ max(各方案等概率期望值)  │
└────────────┴──────────────────┴─────────────────────────┘
```

### 后悔值计算步骤 ★★★★★
```
① 找出每种状态下所有方案的最大损益值
② 后悔值 = 该状态最大值 - 本方案损益值
③ 计算每个方案在各状态下的最大后悔值
④ 选择最大后悔值最小的方案
```

### 决策树法结构 ★★★★★
```
         [□ 决策点]
            │
    ┌───────┼───────┐
    ↓       ↓       ↓
  方案1   方案2   方案3
    │       │       │
    ↓       ↓       ↓
  [○ 状态点]  ...  ...
    │
  ┌─┴─┐
  ↓   ↓
 P₁   P₂
  │   │
  ↓   ↓
 V₁   V₂
```
• 方框 = 决策点（引出方案枝）
• 圆圈 = 状态结点（引出概率枝）
