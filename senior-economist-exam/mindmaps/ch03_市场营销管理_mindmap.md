## 🎯 第二节 目标市场战略（STP）★★★★★

### STP三步骤
```
┌─────────────────────────────────────────────────────┐
│                                                     │
│   S ─── Segmentation ──── 市场细分                  │
│            ↓                                        │
│   T ─── Targeting ───── 目标市场选择                 │
│            ↓                                        │
│   P ─── Positioning ─── 市场定位                    │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 市场细分五性（有效细分条件）★★★★
```
┌────────────────────────────────────┐
│         有效细分的"五性"             │
├──────────┬─────────────────────────┤
│ 可衡量性  │  规模可衡量              │
│ 可进入性  │  能进入并服务            │
│ 可盈利性  │  有足够利润空间           │
│ 差异性    │  子市场间有明显区别       │
│ 可行动性  │  企业有能力服务          │
└──────────┴─────────────────────────┘
```

### 三种目标市场策略对比 ★★★★★
```
┌──────────────┬──────────────┬──────────────┬──────────────┐
│    策略       │    含义       │    优点       │    缺点       │
├──────────────┼──────────────┼──────────────┼──────────────┤
│ 无差异营销   │ 单一产品服务整个市场│ 规模经济、成本低 │ 忽视差异、竞争激烈│
│ 差异性营销   │ 不同产品服务不同市场│ 满足多元需求    │ 成本高、管理复杂  │
│ 集中性营销   │ 集中资源服务少数市场│ 专业化、精准    │ 风险集中       │
└──────────────┴──────────────┴──────────────┴──────────────┘
```

## 📦 第三节 市场营销组合策略

### 4P → 4C → 4R 演进 ★★★★★
```
┌─────────────────────────────────────────────────────────────────┐
│                        营销理论演进                               │
├──────────────┬──────────────┬────────────────────────────────────┤
│     4P       │     4C       │              4R                    │
│  (麦卡锡)    │  (劳特朋)    │           (舒尔茨)                │
├──────────────┼──────────────┼────────────────────────────────────┤
│ Product     │ Customer     │ Relevance     (关联)               │
│ 产品        │ 顾客需求      │ 与顾客建立关联                     │
├──────────────┼──────────────┼────────────────────────────────────┤
│ Price       │ Cost         │ Reaction       (反应)              │
│ 价格        │ 顾客总成本    │ 快速反应                            │
├──────────────┼──────────────┼────────────────────────────────────┤
│ Place       │ Convenience  │ Relationship   (关系)               │
│ 渠道        │ 购买便利      │ 长期稳定关系                        │
├──────────────┼──────────────┼────────────────────────────────────┤
│ Promotion   │ Communication│ Reward         (回报)               │
│ 促销        │ 双向沟通      │ 回报                                  │
└──────────────┴──────────────┴────────────────────────────────────┘
         企业视角          消费者视角        关系视角
```

### 产品生命周期四阶段 ★★★★★
```
    导入期 ──→ 成长期 ──→ 成熟期 ──→ 衰退期
      ↑          ↑          ↑          ↑
    销量低      快速增长    销量最大    下降
    成本高      利润上升    竞争激烈    利润减少
      ↓          ↓          ↓          ↓
   快速撇脂    改进品质    市场改良    维持/收缩
   缓慢撇脂    开拓市场    产品改良    放弃
   快速渗透    调整价格    营销组合改良
   缓慢渗透
```

## 💵 第五节 定价策略

### 定价方法三导向 ★★★★
```
┌─────────────────────────────────────────────────────────┐
│                      定价方法                            │
├─────────────┬─────────────────┬─────────────────────────┤
│  成本导向    │    需求导向      │       竞争导向          │
├─────────────┼─────────────────┼─────────────────────────┤
│ 成本加成定价 │ 理解价值定价      │ 随行就市定价            │
│ 目标利润定价 │ 需求差异定价      │ 密封投标定价            │
└─────────────┴─────────────────┴─────────────────────────┘
```

### 新产品定价策略 ★★★★
```
撇脂定价（高价）              渗透定价（低价）
     ↓                            ↓
 短期高利润                  快速占领市场
 快速收回研发成本            提高市占率
 适用于技术壁垒高产品        适用于价格敏感市场
```
