# CLAUDE.md

这个仓库是用户的 Obsidian 第二大脑和长期外部记忆库。

Claude Code 进入本仓库时，请按这个顺序读取：

1. `Memory/INDEX.md`
2. `MOC/第二大脑控制台.md`
3. `Memory/ACTIVE_PROJECTS.md`
4. `Memory/WORKFLOWS.md`
5. `Memory/DECISIONS.md`

只在需要证据时再读取原始笔记目录，例如 `AI/`、`编程/`、`自媒体/`、`金融/`、`商业/`、`生活/`、`Inbox/`。

## 协作约束

- 不要默认全库扫描。先用 `rg` 精准搜索。
- 不要默认移动或删除 `Inbox/` 中的内容。
- 不要把原始长文复制进 `Memory/`，只写稳定结论、长期偏好、项目状态和可复用工作流。
- 如果要批量改标签、合并重复笔记、清理未分类内容，先给出预览和风险说明。

## 记忆写入标准

适合写入 `Memory/` 的内容：

- 用户长期偏好和稳定工作流。
- 反复出现的项目目标。
- 已验证的决策和取舍。
- 对 Claude Code/Codex 有帮助的库结构说明。

不适合写入 `Memory/` 的内容：

- 未验证的传闻。
- 单篇文章摘要。
- API key、token、账号、隐私资料。
- 只适合临时任务的细节。
