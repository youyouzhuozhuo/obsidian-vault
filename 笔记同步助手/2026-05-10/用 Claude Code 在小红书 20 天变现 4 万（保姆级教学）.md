人粉丝群，很快满员。</p><p>免费分享一阵后我在想，能不能把它升级成可视化教练系统，挂橱窗卖。</p><p>定价 99 元，第一天 10 个人买，还有 2 个定制客户。现在 200 多人买了，三个平台 10 天涨粉 2 万，加上一对一咨询、Claude Code 部署、定制 Skill，20 天变现 4 万。</p><p>今天把完整流程拆给你，任何人都能复制。</p><p><img src="https://pbs.twimg.com/media/HH3kTg6bAAEJsjP.jpg" alt="" /></p><h2>这个生意的起点就一句话——</h2><p>找一个受众大的、需求够硬的方向，做低客单价的虚拟产品。</p><p>举几个：雅思备考模板、Notion 模板、大神思维模型整理、TED 演讲稿整理。</p><p>售价 39-99 之间，不要超过 99。</p><p>为什么不能超过 99？超过 99 客户就开始动脑子了——「我真的需要吗」「我能不能自己做」。99 以下基本是冲动消费区，对的人看到价格直接就付了。</p><h2>第一步，选你要做什么。</h2><p>如果你不知道做什么，就从自己上学到上班划过的钱开始捋。你付过的所有课程、训练营、模板、付费社群，每一笔都是潜在题材。你自己付过钱，意味着这个需求是真的。</p><p>随便选一个，去小红书搜，看有没有产品。有的话买一份。</p><p>然后 Claude Code 上场。</p><p>把你买回来的产品扔给 cc，让它洗稿一遍，包装成你自己的。</p><p>最好申请一个版权证明，学猫猫侠那样。然后上架小红书橱窗。</p><p>这一步最麻烦，下期单独写一篇讲。</p><h2>产品有了，下一步铺内容。</h2><p>回到你刚才买产品的那家店。这家店肯定关联了一些博主橱窗，也有自己的账号。看他发什么，直接拿去给 Claude Code 洗稿，发出来。</p><p>要像素级模仿，但不要纯搬运。</p><p>模仿和搬运的边界在哪，后面专门出一期讲。先记一句：模仿结构和角度，内容、案例、表达全部换成你自己的。</p><p>cc 把这件事的效率拉得非常高。纯图文一天能出 5-10 篇，三天做完一周的量。</p><p>剩下的就是按时出摊等爆款，每天花的时间少于一小时。</p><h2>然后是交付。</h2><p>不要直接发文件。给客户一个网盘链接。</p><p>两个原因：</p><p>第一，百度网盘和夸克网盘都有拉新返现，客户为了下载会注册，你顺带赚一笔。150 单算下来多出几百到上千块。</p><p>第二，更重要——网盘文件夹里除了客户买的产品，要放上你其他产品的介绍，还有微信二维码。</p><p>为什么？</p><p>因为我们要的是一个链路收钱，不是一笔生意。</p><p>虚拟产品只是付费的开始，不是付费的结束。</p><h2>后面要跟上的东西：</h2><p>定制服务、供应链、其他产品、用这个产品配套要的工具。</p><p>举几个：</p><p>买 skill 的客户，需要 Claude Code，需要 token。</p><p>买雅思模板的客户，可能需要陪跑、批改、更高阶的 skill。</p><p>买思维模型的客户，可能需要 Notion 模板规划，也可能想要其他大神的思维模型。</p><p>买商业模式文档的客户，可能需要链接供应链——可以组建一个付费社群。</p><p>后面专门出一期，把所有可行的付费链路列出来，按需自取。</p><p>一句话——卖东西要想一整个网络，不是一个单点。</p><p>这一套跑下来，客户就在你微信里了。</p><p>这时候要做的是在视频号发高端产品的视频。为什么是视频号？后面单独讲。</p><h2>最后两个建议：</h2><p>客户少于
