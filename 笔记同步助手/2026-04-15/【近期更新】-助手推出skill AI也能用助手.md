4月13日

1.助手龙虾技能更新，获取云空间数据免费，同时支持用龙虾提交链接了（5 AI积分/链接），欢迎大家试用，安装skill 直接复制下方内容发给AI/龙虾就好

```
安装这个skill
https://clawhub.ai/notesynchelper/notehelper
```

2.如果是微信发给助手的，我们提供结果回调，可以接入龙虾，实现完整的AI 工作流。

例如：转发内容给助手 → 助手处理完成后提醒你的龙虾/Agent → 龙虾/Agent完成后续各种玩法（免费）

```
回调地址
https://obsidian.notebooksyncer.com/settings
```

4月7日

1.支持飞书文档、Lark文档、腾讯文档、钉钉文档、WPS文档（马上马上）等五大文档，转发链接给助手即可保存到自己的笔记本

（OneNote、Obsidian、思源笔记、Notion 都支持）

Obsidian助手 插件功能更新

1.优化了图片/附件本地化功能，新版本支持在本地化的过程中修改笔记

![](https://pic.clipfx.app/3e8f3044b04ce518d66c9979bfb5955a.png)

2.优化了自动同步功能，防止因网络问题导致后续自动同步失败  
3.优化了时间模板变量，笔记和附件可以存到一个文件  
4.附件文件夹路径修改后立即生效，不用重启插件 感谢群友反馈  
5.去掉了容易手滑的resync all 命令  
6.优化了合并消息逻辑，追加合并消息的时候不会影响属性写入  
7.防止高频同步丢消息，优化了处理逻辑避免高频同步时丢消息  
8.可以自动去掉文章中的标签，红薯、小绿书里大量的标签不会干扰Obs标签
