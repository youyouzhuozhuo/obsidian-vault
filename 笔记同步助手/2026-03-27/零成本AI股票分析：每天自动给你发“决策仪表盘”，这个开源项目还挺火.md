股票、期货这些平时最头疼的应该就是盯盘盯到眼花吧？新闻满天飞、K线乱晃、基本面数据一大堆，到底买还是卖？很多人靠直觉，或者刷论坛跟风，结果亏得一塌糊涂。最近看GitHub上这个开源项目还挺火：**daily\_stock\_analysis**，简单说它就是用AI帮你把A股、港股、美股的自选股每天自动分析一遍，生成一份清晰的“决策仪表盘”，直接推送到企业微信、Telegram、飞书或者邮箱。项目目前已经25.7k星、26.6k叉，看来很多人拿来部署了，最近刚更新到v3.10.1（2026年3月24日），活跃度很高，50多位贡献者一起在维护。

也许它真的有用，值得每位想轻松做投资的朋友了解，今天就来看看这个项目。

​

### 先说它到底能给你什么

简单讲，这个工具就像给你配了个“AI投研助理”：

-   • **每天自动出报告**：工作日北京时间18:00左右（可自定义），自动抓取实时行情、新闻、舆情、筹码分布、技术指标（MA、趋势、量能等），然后用LLM综合判断，给出**一句话核心结论 + 精确买卖点位 + 操作检查清单**。
    
-   • **支持三市场**：A股（含科创板）、港股、美股（包括指数如SPX、DJI、IXIC），数据源来自AkShare、Tushare、YFinance等多渠道，比较全面。
    
-   • **决策仪表盘示例**（真实推送风格）：
    


推送还能带大盘复盘、板块涨跌、风险警报和利好催化点，一目了然。

-   • **额外实用功能**：
    

-   ◦ Web界面：本地启动后浏览器打开，能看历史记录、批量管理、Agent“问股”（支持均线金叉、缠论等11种策略，多轮对话）。
    
-   ◦ AI回测：自动验证历史分析的胜率、止盈止损命中率，让你心里有底。
    
-   ◦ 智能导入：支持截图、CSV、剪贴板直接加自选股，支持多模态的模型还能帮你识别提取代码。
    
-   ◦ 多渠道推送：企业微信、飞书、Telegram、钉钉、邮件、Discord……随便挑。
    

最牛的是**零成本部署**：用GitHub Actions就能定时跑，不用买服务器、不用24小时开电脑。这也是这个项目fork叉这么多的原因吧。

### 具体怎么上手？三分钟就能跑起来

项目作者把入门做得非常友好，分两种方式：

**1\. 最推荐：GitHub Actions（5分钟零服务器）**

-   • 点右上角“Fork”仓库（可以顺手给作者点个Star支持一下）。
    
-   • 去Settings → Secrets and variables → Actions，添加几个关键Secret：
    

-   ◦ AI模型Key：直接填Gemini/OpenAI/Claude或其他模型的 Key。
    
-   ◦ 推送渠道：比如企业微信Webhook URL（最简单，几分钟就能配好）。
    
-   ◦ 自选股列表：STOCK\_LIST填“600519,hk00700,AAPL”（逗号分隔）。
    

-   • 开启Actions工作流，默认工作日自动跑。
    
-   • 想马上测试？点“Run workflow”手动触发就能立刻跑一次。
    

**2\. 本地部署（适合想看Web界面的朋友）**

![](https://pic.clipfx.app/9c0a600247c73ef5d692175067f43110.png)

-   • `git clone https://github.com/ZhuLinsen/daily_stock_analysis.git && cd daily_stock_analysis`
    
-   • `pip install -r requirements.txt`
    
-   • 复制`.env.example`改名称为`.env`，填好大模型 Key和推送渠道配置。
    
-   • `python main.py --webui`启动Web界面，浏览器访问[http://127.0.0.1:8000](http://127.0.0.1:8000)就能玩了。
    
-   • Docker部署也有一键脚本，有服务器的可以直接上。
    

配置好后，它会自动帮你抓数据、分析、生成报告、推送。整个流程全自动化，你只需要每天早上看看推送报告就行。

### 它分析得准吗？

**优点很明显：**

-   • **信息整合能力强**。人工看盘容易遗漏，它把技术面、基本面、舆情、资金流一次性揉在一起，用AI给出结构化结论，比自己零散刷雪球强多了。
    
-   • **纪律性好**。每次都带止损点、检查清单，还内置“严禁追高”“乖离率超5%预警”等规则，帮新手少犯低级错误。
    
-   • **回测透明**。它会告诉你历史分析的胜率，你可以自己判断这个模型在当前风格下表现如何。
    
-   • **开源可定制**。想换更强的本地Ollama模型？想加自己的策略？代码全开放，社区活跃，随时能改。
    

**也要说说局限性（这点非常重要）：**

-   • **它不是神**。市场本身充满不确定性，再好的AI也只能基于公开数据做概率判断，不能预测黑天鹅。项目README里写得清清楚楚：**仅供学习研究，不构成投资建议，作者不对任何损失负责**。
    
-   • **依赖外部API**。大模型的基座能力也不容忽视，数据源偶尔也会延迟或缺失（尤其是港美股实时性）。
    
-   • **新手上手有门槛**。配置Secrets对完全没用过GitHub的朋友可能需要10-20分钟适应，但官方有完整文档和FAQ，跟着走基本没问题。
    
-   • **效果因人而异**。如果你自选股选得乱七八糟，或者不结合自己的风险偏好和仓位管理，光看AI报告还是容易冲动。
    

我的建议是：**把它当成“第二意见”**，而不是唯一决策依据。先用小仓位或模拟盘验证一个月，看看回测胜率和实际表现，再决定怎么用。最好结合自己对行业的了解、宏观面判断一起看。投资永远是概率游戏，工具只是帮你把信息处理得更高效，真正决定盈亏的还是你的交易纪律和心态。
